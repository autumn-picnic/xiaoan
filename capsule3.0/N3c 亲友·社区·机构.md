# applicability

## use_when

- 用户想倾诉/求助但面临心理障碍（羞耻、怕不被信、怕激化）或不确定向谁开口
- 用户已尝试求助但遭质疑/劝和/敷衍，不确定下一步
- 用户处于孤立状态，需要建立至少一个外部支撑点
- 用户想了解报警之外的正式求助渠道（职能、流程、限度）
- 用户当前未处于正在发生的暴力现场

## do_not_use_when

- 正在遭遇暴力/紧急人身危险 → 转即时危机
- 自伤风险或表达活不下去 → 转危机干预
- 明确需要保护令具体流程 → 转N5p
- 需要完整离婚法律程序 → 转N5e

# render_policy

- 让用户理解说出来本身就是保护性行为，不需要等到"够严重"才开口
- 强调安全评估的重要性，安全性比亲疏远近更重要
- 碰壁是正常的不是终点，可以换人、换机构、向上一级反映
- 机构求助不会自动失控，大多数介入是温和的（倾听、登记、转介）
- 不编造本地具体资源（地址、电话）；涉及具体地点时提示用户通过12338或当地妇联官网核实本地可用资源

# recognize

- 让一个外部的人知晓情况本身就是保护性行为，施暴者最怕你不再孤立，不需要等到"够严重"才开口
- 选人需要安全评估，施暴者通常对外形象良好，核心问题是这个人会不会转头告诉施暴者，安全性比亲疏远近重要，能听你说、不急着评价比能"帮你解决问题"更重要
- 碰壁是正常的不是终点，一个人的不良反应不代表所有人都会这样，第一个渠道态度敷衍不代表所有渠道都这样，可以换人、换机构、向上一级反映、请亲友陪同再去
- 机构求助不会自动失控，找妇联/社区不会自动启动刑事程序，大多数介入是温和的（倾听、登记、转介），用户可以明确说"我现在只想了解信息请不要联系对方"，大多数机构会尊重
- 即使当下没有立刻"解决问题"，妇联归档的投诉登记表、危险评估量表、回访记录，必要时可依法作为遭受家暴的证据材料——登记本身就在积累证据

# act

## 通用安全原则（适用于所有求助行为）

- 开口前先评估：这个人/渠道会不会让施暴者知道？哪怕是亲妈，如果她和施暴者关系密切，可能不是最安全的第一选择
- 通讯安全：通话记录会被检查时用公共电话、借别人手机或趁对方不在时打，打完删记录；对方监控手机/电脑时不用家中设备搜索求助信息，用单位电脑、朋友手机或直接当面去机构
- 任何求助行为都可能激怒施暴者，求助的同时注意加强自我保护

## 向亲友求助

### 选人原则（安全性 > 亲疏）

- 优先：和施暴者没交集的朋友、你这边家人中比较开明的、信任的同事
- 暂排除：和施暴者关系近的人、过去劝你忍的人、嘴快爱传话的人、容易冲动会去找对方理论的人

### 如何开口

- 用一句话试探："我最近家里有些事情让我很不安，能不能跟你说说？"或"我跟你说一件事，你先别急着给建议，我就是需要有人听"
- 可以说多少就说多少，"我不确定我经历的算不算家暴但我想说出来"这就够了，不需要一次讲清所有，说到一半哭了、说得颠三倒四都正常

### 给对方一个具体请求

- "帮我保密就好"
- "如果我三天没回你消息你主动联系我"
- "帮我保存几张截图我怕被删"
- "如果我突然说要过来你能让我暂住吗"

### 紧急暗号（进阶）

- 和信任的人约定：收到特定表情时代为报警，或约定"借东西"为由敲门以打断暴力

### 该场景特有风险

- 对方可能冲动去找施暴者理论 → 说完后立刻告诉对方："请你什么都不要做，我需要的只是有人知道这件事"

## 向机构求助

### 1.定位你现在最需要什么

- 有人倾听/情绪支持
- 了解选项和信息
- 短期安全住所
- 法律帮助
- 伤情证据保全
- 从最迫切的开始，不需要一次解决所有

### 2.匹配渠道

- 全国妇联热线12338：通用入口，可倾听、告知当地资源、协调当地妇联介入
- 当地妇联（区县级）：可提供心理支持、危险评估、出面谈话、协助报警/保护令/庇护、出具家暴证明、定期回访；
  标准流程为"登记 → 危险评估 → 分级处理 → 结案存档 → 跟踪回访"，不会一上来就联系对方或启动法律程序；
  受害人因受强制或威吓无法自行申请保护令时，妇联可代为申请或指导近亲属申请
- 社工：可提供全程个案服务、心理辅导、链接资源
- 居委会/村委会：可上门劝阻、做家暴情况登记（可作证据）、报告高危个案
- 庇护所/民政救助站：反家暴法第十八条规定县级或设区市级政府可单独或依托救助管理机构设立临时庇护场所，为家暴受害人提供临时生活帮助（通常免费），地址保密
- 医院：就医时告知"家庭暴力致伤"，要求写入病历
- 法律援助12348：为家暴受害人提供免费律师、法律咨询；家暴受害人主张相关权益时不受经济困难条件限制（《法律援助法》第三十二条）
- 公益热线：源众17701242654（工作日）；红枫010-64033383（请拨打前确认服务时间）

### 3.如何开口与边界设定

- 不确定从哪开始时：拨打12338直接说"我遭遇了家庭暴力，需要帮助"
- 明确告知边界："我现在只是想了解信息，请不要联系对方"；"我不是来调解的，我是要做情况登记"

### 4.碰壁后的升级路径

- 第一次效果不好不代表所有渠道都这样
- 可以换人/换机构/向上一级反映/请亲友陪同再去
- 反家暴法第十三条规定相关单位接到投诉后"应当给予帮助、处理"，被推诿时可以引用此条，或向上一级妇联/民政部门反映
- 记录每次求助的时间、接待人、结果，后续投诉或转介时有据可查

### 向机构求助的特有风险

- 机构可能未经同意联系施暴者或启动调解 → 明确说"请不要联系对方"/"我不是来调解的"
- 机构人员也可能敷衍或不专业 → 这不代表你不该求助，换一个渠道继续
