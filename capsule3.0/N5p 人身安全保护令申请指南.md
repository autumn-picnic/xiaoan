# applicability

## use_when

- 用户正在经历或担忧对方的跟踪、骚扰、威胁、接近行为
- 用户有意愿通过法律途径建立物理安全边界
- 用户想了解保护令的申请条件、流程、证据要求
- 用户已有保护令但对方违反需要知道下一步怎么做
- 用户遭受精神暴力（谩骂、恐吓、侮辱、跟踪等）但不确定是否"够格"申请
- 用户对保护令存在误解需要纠偏

## do_not_use_when

- 用户正在遭遇暴力或处于紧急人身危险中 → 应先转至紧急求助模块
- 用户有自伤风险或表达活不下去 → 应先转至危机干预模块
- 用户描述的情况不属于家庭成员或共同生活关系之间的暴力 → 不适用反家暴法

# render_policy

- 让用户理解保护令是一个真实可用、门槛不高的法律工具
- 不需要先离婚才能申请，不需要等到被打了才能申请，不需要有明显外伤，精神暴力同样受保护
- 将申请保护令拆解为清晰的操作流程，让用户知道去哪里、带什么、怎么写、提交后会发生什么
- 提醒申请保护令可能激怒对方，需要同步准备安全撤离计划

---

# recognize

- 面临家庭暴力的现实危险就可以申请，不需要等到被打了
- 申请保护令不以提起离婚等民事诉讼为条件，可以完全独立申请
- 保护范围包括家庭成员以及同居情侣、儿媳/女婿/公婆等共同生活的人
- 法院审查标准为"较大可能性"，无需达到严格诉讼标准，能有多少证据算多少
- 完全免费，不收取任何诉讼费用
- 一般情况72小时内作出裁定，紧急情况24小时内出具
- 违反保护令可处罚款、司法拘留（最高15天），情节严重追究刑责
- 常见误解澄清：先离婚才能申请是错误的，没有严重伤情不能申请是错误的，保护令只是废纸是错误的

# act

## 操作流程

1. 确认管辖法院（免费）：向申请人居住地、被申请人居住地、家庭暴力发生地任一基层法院申请

2. 准备身份与关系证明（必备）：申请人与被申请人的身份证，证明双方关系的材料（结婚证、户口本、同居证明等）

3. 收集家暴证据（能有多少算多少）：报警回执/告诫书/行政处罚决定书，照片（伤痕、打砸现场），病历资料，录音/录像/短信/聊天记录，证人证言，妇联/社区求助记录，对方保证书/悔过书，无法自行收集的可申请法院调查收集

4. 撰写申请书：写明双方基本信息、具体请求（禁止暴力/骚扰/接触/迁出住所等）、事实及理由（具体暴力经过），书写困难可到法院口头申请由工作人员记录成笔录

5. 提交申请：到法院立案庭窗口递交，或委托近亲属/妇联/居委会等代为申请，部分法院支持线上递交

6. 等待裁定与执行：72小时内（紧急情况24小时内）作出裁定，裁定后送达双方并抄送辖区派出所和居委会/村委会，有效期最长6个月，到期前可申请延长

7. 对方违反保护令时：立即报警并告知你持有保护令，记录违反事实（拍照、录像、保留通话记录），违反者将面临罚款、拘留或刑事追责
