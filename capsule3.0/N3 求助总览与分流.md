# triggers

- "我该怎么办"
- "我不知道从哪里开始"
- "我需要帮助"
- "我想求助"

# applicability

## use_when

- 用户首次求助、不知从何开始
- 用户想了解可以采取哪些行动
- 用户需要被引导聚焦到具体方向

## do_not_use_when

- 用户正在遭遇暴力或当前不安全 → 转【即时危机】
- 用户已明确知道自己需要什么(如直接询问报警、法律等具体问题) → 转对应胶囊(本胶囊 act 中的方向性引导仅用于对话过程中用户追问,不用于入口判断)

# render_policy

- 本胶囊是"地图"而非"急救手册":功能是帮用户看清方向,不替代具体胶囊给完整方案
- 语气:先肯定求助行为,再聚焦方向
- 输出约束:根据分诊结果给2-3个最相关方向,不要一次铺开全部选项
- 当多个方向同时相关时,优先处理安全相关的,再回应用户主动提出的问题
- 不把报警作为默认首选建议;报警作为唯一选项可能加剧危险

# recognize

- 愿意开口求助本身已经是重要的一步,很多人在这一步之前已经独自承受了很久
- 安全 → 支持 → 法律行动;顺序可以根据处境灵活调整,但安全始终优先
- 情绪缓和、距离隔开之后110介入才更有效、更安全
- 有人陪伴会让后续每一步都更好走,打破孤立本身就是行动
- 用户说"我该怎么办"时,可能是在寻求行动指引,也可能是情绪崩溃后的无力表达——需要先判断再分流

# act

## 分诊步骤

1. 判断用户此刻的主要状态:是需要情绪承接,还是在寻找行动方向?(如果不确定,先回应情绪,再问"你现在最想解决的是什么?")
2. 确认安全:"你现在安全吗?对方在你身边吗?"
3. 根据回答进入对应条件块;如果回答模糊,用兜底策略追问,不猜测
4. 每次只给出1-2个最相关方向,给完后问用户是否需要了解其他方面

## 条件块

### 如果用户此刻主要需要情绪承接

- 不急于分流,先回应感受:"你愿意说出来,这本身就很不容易。"
- 允许用户表达,不打断、不催促进入"下一步"
- 情绪稍缓后再问:"你现在最担心的是什么?"引导进入具体方向
- 如情绪强烈且持续,可转接【心理支持】

### 如果对方此刻在身边或用户当前不安全

- 确认是否有即时人身威胁 → 有,转【即时危机】
- 无即时威胁但对方在场:建议找借口离开对话("你的安全最重要,如果现在不方便,随时可以回来"),告知可以在安全时再联系
- 同住但对方暂时不在:当前有行动窗口,根据用户需求进入其他条件块

### 如果有伤情

- 尽快就医;就医记录本身即证据,之后不需要重新验伤
- 有伤情时拨打120是合理选择(医护会到场,就医记录兼具证据功能)
- 不一定要"先报警再就医",顺序可以反过来
- → 可详细查看【就医】【证据收集】

### 如果已脱离对方但不知下一步做什么

- 优先确认落脚点是否安全、是否有人知情
- 如果无安全落脚点 → 引导至庇护资源
- 如果完全孤立、无人知情 → 告诉至少一个信任的人;打破孤立本身就是安全措施;如果身边没有可信任的人,机构也可以(妇联、社工站、热线)
- → 可详细查看【亲友·社区·机构求助】

### 如果对话中用户问"要不要报警"

- 报警不是唯一选项,也不一定是第一步
- 更适合在情绪缓和、距离隔开后进行
- 有生命危险时应立即110
- 事后去派出所做笔录同样有效,不需要"当场"报警
- 派出所实践中可作为临时躲避场所直接走入,不一定需要正式报案(注:实践层面可行,非法定权利,个别情况可能被拒)
- → 可详细查看【报警】

### 如果想了解法律能做什么

- 可以申请人身安全保护令、告诫书(不需要先离婚、不需要先报警)
- 证据越早开始收集越好,但现在开始也来得及
- → 可详细查看【保护令】或【出具告诫书】

### 如果纠结去留

- 不需要现在做决定
- 可以先了解"如果离开,需要准备什么",了解不等于决定
- → 可详细查看【安全准备】或【是否离婚】

## 兜底

如果用户的表达不明确落入任何条件块:

- 不要猜测,追问一个具体问题:"你现在最担心的是什么?"或"你现在安全吗?"
- 避免一次输出所有方向
- 用户回答后重新进入条件块匹配
