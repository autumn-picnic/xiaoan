# applicability 适用规则

do_not_use_when:
- 用户正在遭遇暴力或处于紧急人身危险中（应先转至紧急求助）
- 用户有自伤风险或表达活不下去（应先转至危机干预）

# recognize 认知引导
recognize.purpose: "打破'家务事'迷思，帮助用户理解警察推诿是执法漏洞而非法律事实，确认自己有权要求依法履职。"

content:
- key_point: "'家务事'是过时的执法借口，不是法律事实"
  message: "自2016年《反家庭暴力法》实施以来，家庭暴力是违法行为，不是家务事。公安机关接到家暴报案后必须及时出警、制止暴力、调查取证。"

- key_point: "你不需要'完美受害者'才配报警"
  message: "只要发生暴力，你就有权要求警方依法履职。恐惧和犹豫是正常的，但沉默会让暴力升级——多项研究表明首次家暴未受有效法律干预，复发率极高。"

- key_point: "警察推诿可能是程序疏忽，也可能是失职"
  message: "'你们夫妻吵架我们管不了'、'没打伤就不算事'——这些回应违反《反家庭暴力法》第15条，属于行政不作为，你可以投诉。"

- key_point: "常见推诿情况的法律定性"
  message: "①'这是家务事'/拒绝受理→违反《反家暴法》第15条；②劝说'回去好好过'→暴力是违法行为，不是民事问题；③以'影响孩子档案'威胁→行政拘留不留案底，这是常见劝退话术；④要求双方到场调解→调解须自愿，可拒绝。"

# act 行动指引
act.purpose: "提供低负担、高策略的应对步骤，帮助用户在当场及事后有效回应推诿，并在必要时绕过警方直接寻求司法保护。"

steps:
1. 报警时的黄金话术（现场控制）：大声清晰说："我是受害人，我正在遭受家庭暴力，对方殴打/威胁我，我的人身安全受到现实危险。我要求立即出警，制止暴力，并依法调查取证！"——避免说"我们吵架了"、"他脾气不好"，必须说"被打"、"暴力"、"人身安全受威胁"。全程开启录音，记录警察回应。
2. 现场取证与文书固定：①索要《接处警记录》（须含时间、地点、当事人、简要案情）；②索要《家庭暴力告诫书》（依据：《反家暴法》第16条，话术："根据《反家庭暴力法》第16条，情节较轻也应出具告诫书，请依法开具并送达给我和居委会"）；③要求民警拍摄现场、伤情、破损物品。如警察拒绝出具告诫书，说："如果您认为不需要出具，请在笔录中写明理由，否则我将视为行政不作为"，并当场拨打12389投诉。
3. 事后补救与投诉：①12389（公安机关违纪违法举报平台，最直接）；②12345（市长热线，转办至政法委或公安局督察）；③向当地检察院申请立案监督或行政违法行为监督；④联系当地妇联（12338），请求其向公安机关发出支持意见。投诉话术："我于X月X日X时在XX地报警遭家暴，出警民警以'家务事'为由拒绝出警/拒绝制作笔录/拒绝出具告诫书，违反《反家庭暴力法》第15条，要求督察部门介入调查。"
4. 升级法律程序（绕过警方推诿）：即使警方未出具告诫书，只要有报警记录（哪怕只是接警单）、伤情照片、录音，即可直接向法院申请人身安全保护令（→N5p）。如果警方明确拒绝出警或拒绝履行职责，可向法院提起行政诉讼，告公安机关"行政不作为"。

scripts:
- 报警时："我是受害人，我正在遭受家庭暴力，对方殴打/威胁我，我的人身安全受到现实危险。我要求立即出警，制止暴力，并依法调查取证！"
- 索要告诫书："根据《反家庭暴力法》第16条，情节较轻的也应出具告诫书。请依法开具，并送达给我和居委会。"
- 警察拒绝时："如果您认为不需要出具，请在笔录中写明理由，并告知我法律依据。否则我将视为行政不作为。"
- 投诉时："我于X月X日报警遭家暴，民警以'家务事'为由拒绝制作笔录/出具告诫书，违反《反家庭暴力法》第15条，要求督察部门介入。"

safety_note:
- 不与警方正面冲突，但坚持权利；目标优先，不被动摇。
- 一条路不通换一条——派出所推诿时，同步向12389投诉并直接向法院申请保护令，不要等投诉结果。
- 不同场景：第一次报警→拒绝私下和解，坚持出具告诫书；多次报警无效→从派出所升级到分局督察再到市检察院；担心报复→先申请保护令再报警。

# ground 依据与限制
ground.purpose: "提供警方法定职责的法律依据，帮助用户了解自己的权利，同时诚实说明执法差异和证据局限。"

legal_basis:
- 《反家庭暴力法》第15条：公安机关接到报案后应当及时出警，制止暴力，调查取证，协助就医、鉴定（强制出警权，警察无权以"家务事"拒绝出警）
- 《反家庭暴力法》第16条：情节较轻不处罚的，应出具告诫书（即使未造成伤害，只要有家暴行为，就应出具）
- 《反家庭暴力法》第35条：未报案造成严重后果的，责任人受处分（追责权）
- 《治安管理处罚法》第73条（2025修订）：拒不执行禁止家暴告诫书的，处拘留、罚款
- 《治安管理处罚法》第51条：殴打他人处拘留罚款；殴打孕妇、老人、儿童从重
- 《妇女权益保障法》第65条：有关部门应预防和制止家暴，提供救助（多部门联动义务）
- 《公安机关办理行政案件程序规定》：家暴案件不得调解（除非受害人自愿且情节轻微），必须调查

practice_basis:
- 多项研究和实务经验表明，若首次家暴未受到有效法律干预，复发率极高（通常估算在60%-80%），极易升级为严重伤害
- 警察说"家务事"属于错误的执法惯性，在一线城市执法规范，偏远地区可能仍存在"和稀泥"现象
- 2026年检察公益诉讼已覆盖家暴领域，可向检察院申请行政违法行为监督

limits:
- 如果没有任何证据（无伤、无录音、无证人），警方可能难以立案——但即使没伤，也要坚持要求出警和记录，记录本身就是证据
- 地域差异：一线城市执法规范，偏远地区可能仍存在"和稀泥"
- 家暴案件原则上不适用调解；如果必须调解，必须在安全、自愿前提下，且不能免除施暴者法律责任
- 投诉和诉讼需要时间，建议同步进行：一边投诉警方，一边向法院申请保护令

route.purpose: 当警方推诿时，用户需要了解如何坚持权利、如何升级投诉，以及绕过警方的替代方案。
