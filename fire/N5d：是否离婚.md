# applicability 适用规则
use_when:
- 用户在考虑离婚，但因恐惧、内疚、经济依赖、施暴者威胁或信息不足而犹豫
- 用户对离婚有误解（如认为必须对方同意、必须有证据、家暴不够严重不能离）
- 用户对离婚有不切实际的期待或恐惧
- 用户在"走"和"留"间摇摆，需要梳理真实需求和现实条件

do_not_use_when:
- 用户正遭遇暴力或处于紧急人身危险中（转NC紧急求助）
- 用户已明确决定离婚，仅需了解具体法律流程（转N5e）
- 用户有自伤风险或表达活不下去（转危机干预）
- 用户仅需情绪疏导而非决策支持
- 仅为一般婚姻冲突，不涉及暴力/控制/虐待

ai_stance:
  core_principle: "AI 不替用户做决定，尊重其在充分信息下的自主选择。"
  prohibited:
    - 不推动离婚：决定权在用户
    - 不做道德评判：绝不暗示"不离婚就是软弱/不够清醒"
    - 不流露失望：对用户选择暂时留下保持尊重
    - 不美化离婚：不将离婚简单等同于"解脱"或"问题自动消失"

# recognize 认知引导
recognize.purpose: "帮助用户理解：决定权在自己，没有标准答案。协助看清被情绪和误解遮蔽的现实，打破认知障碍，建立对离婚的合理预期。"
content
- 决定可以等，准备不需要等。了解选项和评估处境本身就是有意义的
- 犹豫不代表软弱，离开涉及风险，反复摇摆是合理的自我保护
- 间歇性示好是暴力循环的特征，缺乏长期专业矫治的承诺不可信
- 目睹家暴对孩子伤害极大，无暴力的单亲环境优于暴力的完整家庭
- 分居/离婚阶段是暴力升级高危期，安全规划重于仓促行动
- 留下不等于接受暴力，可同时持续建立安全网和保存证据

# act 行动指引
act.purpose: "帮用户建立决策框架：评估处境、厘清需求、盘点资源、了解选项，为自主选择做好准备。"

steps:
1. 评估当前处境：诚实自问——暴力是否升级？是否持续恐惧？他有无致命威胁或全面控制？孩子是否受影响？
2. 厘清真实需求：列出想通过离婚解决的具体问题，评估离婚是否真能解决，或需要哪些配套措施。
3. 多维度评估准备程度：盘点①人际支持网络；②独立养育能力；③经济独立程度；④离婚后暴力再发风险。
4. 了解法律选项：中国诉讼离婚无需对方同意，家暴是法定离婚事由。你有权争取财产、赔偿、抚养权及保护令（→N5e、→N5p）。
5. 评估离开风险：预判对方知情后的反应及暴力升级可能，确保有安全去处和防范措施。
6. 悄悄开始准备（无论走留）：备份重要证件、保存家暴证据、摸清家庭财产负债、建立独立收入来源、寻找信任支持（→N3c）。

common_questions:
- "离婚需要对方同意吗？" → 协议离婚需要。若对方不同意，可直接向法院起诉离婚。家暴是法定事由，查实后法院应判离。
- "第一次起诉法院会判离吗？" → 若无充分证据，第一次可能不判离；但有家暴证据应判离。即使未判，6个月后二次起诉通常会判离。
- "有30天离婚冷静期吗？" → 冷静期仅限协议离婚。家暴案件建议直接诉讼离婚，不受冷静期限制。
- "他说离婚就不让我见孩子/要自杀" → 抚养权按"最有利未成年人"判定，家暴是不利因素。自杀威胁是情感操控，不应成为你的枷锁。
- "我没收入，也没家暴证据，能离吗？" → 无证据也可起诉。无收入者有权分割共同财产并申请赔偿/补偿。
- "离婚后他借探视孩子骚扰我怎么办？" → 可向法院申请中止探视、要求第三方监督探视，或申请保护令禁止骚扰。

scripts

safety_note
- 绝不在无安全计划或激烈冲突时当场提离婚，极易引发暴力升级
- 若对方已有致命威胁，提离婚前应先申请人身安全保护令
- 所有查询（含搜索记录）和准备工作必须在对方不知情下进行

# ground 依据与限制
ground.purpose: "提供法律依据，说明实务困难，不替用户做决策。"

legal_basis
- 《民法典》第1079条（家暴为法定离婚事由）
- 《民法典》第1077条（冷静期仅限协议离婚）
- 《民法典》第1091条（家暴损害赔偿）
- 《民法典》第1087、1084条（财产与抚养权原则）
- 《民法典婚姻家庭编解释(一)》第63条（首次未判离分居一年后再次起诉应判离）
- 《反家庭暴力法》（保护令制度）

practice_basis
- 受害者平均尝试7次才能最终离开，反复是正常过程
- 分居/离婚阶段是暴力致死高危期，"安全离开"重于"立刻离开"
- 40%-70%家暴案在离婚后暴力未完全停止，需配套保护措施
- 施暴者无专业矫治的改变承诺极难兑现；目睹家暴对儿童伤害极大

limits
- 本胶囊不替用户做决定，不推动离婚，不对经济依赖者简单劝离
- 离婚不是万能方案，不能自动消除后续骚扰风险
- 仅覆盖"决策阶段"，具体诉讼流程转N5e，心理创伤需专业心理咨询
