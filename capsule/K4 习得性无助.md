# 2. 触发条件
triggers:
- "我知道应该离开但就是没有勇气"
- "为什么我多次想求助却始终无法走出这段关系"
- "我觉得自己太软弱了，总是忍受暴力"
- "我试过反抗但还是回到他身边怎么办"
- "我试过报警/找妇联，但没有用，放弃了"
- "我觉得我这辈子也就这样了，认命了"

# 3. applicability 适用规则
use_when:
- "用户表现出强烈的无力感、自我否定，认为任何反抗或求助都无法改变现状"
- "用户因‘反复逃离又反复返回’的行为模式产生深度自责与羞耻感"
- "当前无即时肢体暴力冲突（处于冷战期或日常压抑期）"

do_not_use_when:
    "用户正在遭遇暴力或当前人身安全受到直接威胁",
    "用户的手机、通讯或财务账户正被对方监控",
    "用户表达自伤、自杀风险或出现极度绝望情绪",
    "需要立即报警或寻求紧急庇护的情况"
    
    
# 3. recognize 认知引导
recognize.purpose: 
- "【去病理化与去标签】严禁直接对用户说‘你这是习得性无助’。将此概念作为 AI 的底层诊断，转化为高同理心的脱罪语言。"
- "【对抗羞耻感】核心任务是解除受害者的羞耻感。明确告知对方：‘不离开’、‘又回去了’不是因为软弱，而是大脑在长期高压下的自我保护机制。"
- "【合理化犹豫】肯定受害者过往付出的所有努力（即使失败了），重新定义‘无力’——这是环境的残酷导致的，而不是你的无能。"

content:
  "习得性无助不是性格缺陷或软弱，而是长期受控后的正常心理创伤反应",
  "在重复的暴力循环中，大脑会形成'无论做什么都无法改变结果'的认知模式",
  "这种心理状态与抑郁症的神经机制相似，需要通过外部支持和专业干预来打破",
  "受害者的'不离开'往往有现实障碍（经济、子女、安全），而不仅是心理问题",
  "恢复是一个过程，需要时间、支持和具体的小步骤行动，不需要自责"
