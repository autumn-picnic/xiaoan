# 2. 触发条件

triggers:
  - "我决定离婚了，怎么操作" / "诉讼离婚怎么走"
  - "协议离婚和起诉离婚有什么区别"
  - "他不同意离婚怎么办" / "第一次起诉没判离怎么办"
  - "离婚需要准备什么材料" / "要去哪个法院起诉" / "起诉离婚需要多久"
  - "孩子抚养权怎么争取" / "财产怎么分" / "他转移财产了怎么办"
  - "家暴离婚能多分财产吗" / "他有家暴但我没有证据怎么办"
  - "离婚诉讼期间他来骚扰我怎么办" / "离婚后他还骚扰我怎么办"
  - "我请不起律师" / "离婚协议怎么写"

  以下场景符合其一即可调用：
  - 用户已决定离婚，需了解具体法律流程和操作步骤。
  - 用户想了解协议离婚和诉讼离婚的区别及适用条件。
  - 用户需在家暴背景下争取有利的抚养权、财产分割、损害赔偿。
  - 用户在诉讼中遇到困难（对方不同意、不判离、转移财产、诉讼期间受骚扰）。
  - 用户经济困难需要法律援助。
  - 用户想了解离婚后的法律保护。


# 3. 适用规则


do_not_use_when:
  - 用户正遭遇暴力或处于紧急人身危险中（转NC紧急求助）
  - 用户仍在犹豫是否离婚（转N5d）
  - 用户有自伤风险或表达活不下去（转危机干预）
  - 用户情况不涉及家暴，仅为一般离婚咨询（超出服务范围，建议咨询律师）

# 4. 认知引导（recognize）


recognize:
  content:

    # ———— 模块A：法律站在你这一方 ————

    - theme: "家暴背景下离婚，法律给你的保护"
      points:
        - title: "家暴是法定离婚事由"
          detail: "《民法典》第1079条：实施家庭暴力，经调解无效，应当准予离婚。家暴本身就是感情破裂的法定证据，你不需要另行证明'感情破裂'。"

        - title: "受害方有权请求离婚损害赔偿"
          detail: "《民法典》第1091条：家暴属法定五种过错情形之一。无过错方可请求物质+精神损害赔偿，独立于财产分割。可在离婚诉讼中一并提出，也可离婚后一年内单独起诉。（操作细节见act.damage_compensation）"

        - title: "财产分割照顾无过错方"
          detail: "《民法典》第1087条：离婚财产分割照顾无过错方。施暴者有过错，你可争取多分。这与损害赔偿是两个独立制度，可同时主张。"

        - title: "施暴行为不利于获得抚养权"
          detail: "法院遵循'最有利于未成年子女'原则。有家暴行为的一方通常不被视为适合直接抚养。（争取策略见act.custody_guidance）"

        - title: "离婚诉讼不需要对方同意"
          detail: "这是法定权利。即使TA拒绝签字、拒绝出庭，法院可缺席判决。不配合可能拖延时间，但无法阻止最终离婚。"

    # ———— 模块B：建立合理预期（核心模块）————

    - theme: "诉讼离婚可能遇到的现实困难——尤其是举证"
      framing: "以下信息不是为了吓退你，而是让你做好准备。知道困难在哪里，才能有针对性地克服。每一个困难后面都有你可以采取的行动。"

      points:
        - title: "证据是离婚诉讼中最关键的环节"
          detail: "千千事务所研究显示：仅29.8%的原告提交了证据，超半数为孤证。大多数受害者走进法庭时几乎'空手'——不是因为家暴不存在，而是取证意识不足或时机已过。（完整数据见ground.practice_basis）"
          action: "→ 现在开始保存证据还来得及。证据越充分，胜算越大（→N4）。"

        - title: "第一次起诉不一定判离"
          detail: "即使认定家暴，仍有约24%首次未判离。这是中国司法常见情况。6个月后可再次起诉，第二次有充分证据且分居满一年的应判离。第一次没成功是暂时挫折，不是终点。"
          action: "→ 做好可能需要两次诉讼的心理准备和时间规划。"

        - title: "诉讼需要时间"
          detail: "简易程序3-6个月，普通程序6-12个月。若第一次不判离，全程可能1.5-2年。对方可能拖延、转移财产，应提前申请财产保全。"

        - title: "诉讼期间可能面临报复骚扰"
          detail: "施暴者得知被起诉后暴力可能升级，这是统计上的高危期。"
          action: "→ 应同时申请人身安全保护令（→N5p）。"

    # ———— 模块C：协议离婚 vs 诉讼离婚 ————

    - theme: "两种离婚方式的选择"
      points:
        - title: "协议离婚"
          detail: "双方自愿、对财产和子女达成一致，到民政局办理，有30天冷静期。优点：快速、成本低。家暴背景下风险：协商中权力不对等，受害方易被胁迫让步；施暴者可能利用冷静期反悔或施压。"
          guidance: "如对方愿意且你能确保谈判安全和公平，可选。如存在胁迫或不公平条款，应选诉讼。"

        - title: "诉讼离婚"
          detail: "一方不同意或无法就财产/子女达成一致时，向法院起诉。家暴案件中更常见也更安全——有法院介入，权力不对等能在一定程度上被制度性校正。"
          guidance: "家暴案件建议优先考虑诉讼离婚，尤其是：对方不同意离婚、需主张损害赔偿、担心协商安全。"

# 5. 行动指引（act）
# 

act:

  # ———— 诉讼离婚操作流程 ————

  litigation_steps:
    - step: 1
      name: "前期准备（诉讼前）"
      actions:
        - "确认管辖法院：通常为被告住所地基层法院。因家暴离开住所地连续居住一年以上，可在居住地起诉。"
        - "收集和固定证据：最关键环节。→N4。"
        - "梳理财产状况：列出所有共同财产，尽可能获取财产证据（房产证、银行流水、工资单等）。"
        - "考虑抚养权诉求：整理主要照料人证据（接送记录、学校联系人、医疗记录等）。"
        - "咨询律师（强烈建议）：家暴案件复杂性决定专业帮助的重要性。经济困难可申请法律援助（→N6）。"
        - "安全规划：不要在无安全保障时告知施暴者离婚计划。考虑同步申请保护令（→N5p）。"
      evidence_reminder: "70%的原告走进法庭时没有提交任何家暴证据。即使现在不足，只要没开庭，窗口就没关闭。→N4。"

    - step: 2
      name: "立案"
      actions:
        - "准备起诉材料：起诉状（诉请：判离+财产分割+抚养权+损害赔偿）、身份证、结婚证、证据材料。"
        - "向法院提交：可现场或网上立案。"
        - "缴纳诉讼费：离婚案件50-300元；涉财产分割按标的额另计。经济困难可申请缓交/减免。"
        - "如担心转移财产：立案时同步申请财产保全。"

    - step: 3
      name: "等待开庭"
      actions:
        - "法院送达传票给被告（你不需要自己通知对方）。"
        - "被告可能答辩或反诉——正常程序，不必恐慌。"
        - "继续补充证据：证据交换前均可补交。→N4。"
        - "与律师沟通庭审策略。"

    - step: 4
      name: "开庭审理"
      actions:
        - "庭审流程：法庭调查→举证质证→法庭辩论→最后陈述。"
        - "法院通常先调解：条款公平可接受（调解书=判决书效力）；不公平有权拒绝，继续等判决。"
        - "出庭注意：如实陈述、冷静应对。害怕面对施暴者可提前告知法官，部分法院可安排分开候审或远程出庭。"

    - step: 5
      name: "判决"
      actions:
        - "判决离婚：执行判决内容。对方不执行可申请强制执行。"
        - "判决不离婚（第一次常见）：6个月后可再次起诉，继续收集证据。第二次判离概率显著提高。"
        - "不服判决：收到判决书15日内可上诉。"

  # ———— 离婚损害赔偿操作要点 ————

  damage_compensation:
    purpose: "在家暴背景下主张离婚损害赔偿的操作要点（法律依据见recognize模块A第2点）"
    key_points:
      - point: "提出时机"
        detail: "可在离婚诉讼中一并提出（诉请中列明赔偿金额），也可离婚后一年内单独起诉。建议一并提出，节省成本。"

      - point: "举证要求"
        detail: "需证明：(1)对方存在家暴行为；(2)家暴导致离婚；(3)你遭受的损害（物质+精神）。核心：赔偿能否获支持取决于家暴能否被认定，认定核心在证据。千千事务所数据：360例中仅8人提出，3人因证据不足被驳回。→充分证据是前提（→N4）。"

      - point: "赔偿金额"
        detail: "法律未规定固定金额，法院综合酌定。实务中精神损害赔偿数千至数万元不等。物质赔偿需提供损失证据（医疗费单据、误工证明等）。"

  # ———— 抚养权争取要点 ————

  custody_guidance:
    points:
      - point: "核心原则"
        detail: "法院遵循'最有利于未成年子女'原则。2周岁以下原则上母亲抚养；2-8周岁综合考量；8周岁以上尊重孩子意愿。"

      - point: "家暴因素的影响"
        detail: "施暴者暴力行为（无论是否针对孩子）不利于获得抚养权。法院考虑：暴力环境对孩子的影响、施暴者情绪控制能力、是否存在对孩子的暴力。"

      - point: "有利因素"
        detail: "主要照料人事实、稳定居住和收入条件、孩子情感依附、对方暴力记录。"

      - point: "探视权风险提示"
        detail: "离婚后施暴者通过探视权持续控制是常见问题。可请求'监督探视'（第三方在场或指定场所），降低风险。"

  # ———— 财产分割要点 ————

  property_guidance:
    points:
      - point: "基本原则"
        detail: "共同财产原则上均等分割，家暴案件中无过错方可主张多分。"

      - point: "防止转移财产"
        detail: "施暴者诉讼前/中转移隐匿财产常见。对策：尽早申请财产保全；起诉前尽可能掌握对方财产线索。"

      - point: "法律援助"
        detail: "经济困难的家暴受害者可申请法律援助。2022年《法律援助法》实施后，多地已将家暴受害者纳入免审查经济条件的特殊群体（→N6）。"
