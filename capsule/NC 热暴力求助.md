# triggers

- "他正在冲我发火骂脏话、砸东西，我该怎么办"
- "老公现在又在精神折磨我，一直骂我没用、威胁我，我好害怕"
- "他刚刚又揍了我一顿，说要让我好看，我现在不敢待家里"

# applicability

## use_when

- 用户描述正在遭受或刚刚遭受激烈身体暴力、言语攻击、辱骂、吼叫等暴力行为
- 用户当前处于高压、恐惧或崩溃状态
- 用户寻求立即应对正在发生的暴力行为的办法

## do_not_use_when

- 用户未处于热暴力状态

# render_policy

- 确保在热暴力正在发生时，系统优先推送安全撤离指引而不是常规规划建议，防止用户在危险环境中继续依赖AI对话
- 帮助用户在热暴力正在发生时，立即采取保护行动，优先脱离危险环境，并引导转向专业危机干预机构，而非继续使用本产品作为主要支持
- 本产品不能替代紧急求助热线和专业庇护服务

# recognize

- 热暴力造成的风险是非常即时且严重的，可能快速升级为身体暴力，需要尽快离开暴力环境
- 保护自己及身边人的安全是当前唯一最重要的事，其他所有事情都要放在之后。基本原则是先离开当前暴力环境，再处理后续问题


# act

- 立即判断当前风险性：如果暴力持续甚至升级的风险，立刻找机会离开现场
- 前往安全地点：去信任的亲友家、社区临时庇护点或外出到公共安全场所
- 用安全方式联系帮助：使用他人手机或外出后拨打12338妇女维权热线或110
- 向专业机构求助：描述正在发生的暴力情况，请求紧急指导和资源支持
- 脱离危险并稳定后，再逐步考虑证据收集、法律程序和长期安排
