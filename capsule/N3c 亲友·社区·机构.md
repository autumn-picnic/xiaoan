# fits_any
- 用户想倾诉但不确定向谁、如何开口，或对"说出去"有心理障碍（羞耻、怕不被信、怕激化）
- 用户已告诉亲友但遭质疑或劝和，不知道下一步
- 用户处于孤立状态，需要建立至少一个人际支撑点
- 用户想了解报警之外的正式求助渠道（职能、流程、限度）
- 用户首次求助效果不好，不确定是否继续

# applicability
use_when:
- 用户当前未处于正在发生的暴力现场

do_not_use_when:
- 正在遭遇暴力/紧急人身危险 → 转【即时危机】
- 自伤风险或表达活不下去 → 转【危机干预】
- 明确需要保护令具体流程 → 转N5p
- 需要完整离婚法律程序 → 转N5e

# recognize

1. **说出来的意义。** 让一个外部的人知晓情况本身就是保护性行为——施暴者最怕你不再孤立。不需要等到"够严重"才开口。

2. **选人需要安全评估。** 施暴者通常对外形象良好。核心问题：这个人会不会转头告诉施暴者？安全性比亲疏远近重要。能听你说、不急着评价，比能"帮你解决问题"更重要。

3. **碰壁是正常的，不是终点。** 一个人的不良反应不代表所有人都会这样。第一个渠道态度敷衍不代表所有渠道都这样。可以换人、换机构、向上一级反映、请亲友陪同再去。

4. **机构求助不会自动失控。** 找妇联/社区不会自动启动刑事程序。大多数介入是温和的：倾听、登记、转介。用户可以明确说"我现在只想了解信息，请不要联系对方"——大多数机构会尊重。

# act

## 向亲友开口

1. **评估安全的人。** 优先：和施暴者没交集的朋友、你这边家人中比较开明的、信任的同事。暂排除：和施暴者关系近的人、过去劝你忍的人、嘴快爱传话的人、容易冲动会去找对方理论的人。
2. **用一句话试探。** "我最近家里有些事情让我很不安，能不能跟你说说？"或"我跟你说一件事，你先别急着给建议，我就是需要有人听。"
3. **说出来。** 可以说多少就说多少。"我不确定我经历的算不算家暴，但我想说出来"——这就够了。不需要一次讲清所有，说到一半哭了、说得颠三倒四都正常。
4. **给对方一个具体请求。** "帮我保密就好"/"如果我三天没回你消息你主动联系我"/"帮我保存几张截图我怕被删"/"如果我突然说要过来你能让我暂住吗"。
5. **建立紧急暗号（进阶）。** 和信任的人约定：收到特定表情时代为报警，或约定"借东西"为由敲门以打断暴力。

## 向机构求助

1. 确定现在最需要什么（有人倾听？了解选项？安全住所？法律帮助？伤情证据？），从最迫切的开始。
2. 不确定从哪开始 → 12338是通用入口，直接说"我遭遇了家庭暴力，需要帮助"。
3. 明确告知边界："我现在只是想了解信息，请不要联系对方"；"我不是来调解的，我是要做情况登记"。
4. 第一次效果不好 → 换渠道、向上一级反映、请亲友陪同再去。

## 安全提示

- 开口前先评估：这个人会不会转头告诉施暴者？哪怕是亲妈，如果她和施暴者关系密切，可能不是最安全的第一选择。
- 通话记录会被检查时：用公共电话、借别人手机、或趁对方不在时打，打完删记录。
- 对方监控手机/电脑时：不用家中设备搜索求助信息——用单位电脑、朋友手机、或直接当面去机构。
- 说完后如果担心对方冲动去找施暴者理论，立刻告诉对方"请你什么都不要做，我需要的只是有人知道这件事"。
- 任何求助行为都可能激怒施暴者——求助的同时注意加强自我保护。

# ground

## 法律依据
- 《反家庭暴力法》第13条：受害人及近亲属可向单位、居委会、妇联等投诉求助，相关单位应当给予帮助。
- 《反家庭暴力法》第18条：县级以上政府可设立临时庇护场所。
- 《反家庭暴力法》第23条：受害人无法自行申请保护令时，近亲属、公安、妇联、居委会等可代为申请。

## 实践依据
- 受害者首次披露时获得的反应（被相信 vs 被质疑）显著影响后续求助行为——这是为什么"选对第一个人"很重要。
- 支持网络的建立需要策略性规划，不是"告诉越多人越好"——错误的披露可能导致信息泄露给施暴者。

## 局限
- 居委会/妇联的调解不具法律强制力，施暴者可能不配合。
- 庇护所资源各地差异大，部分地区无独立场所或床位有限。
- 妇联和社区不能替代公权力干预——提供的是协调、服务性工作。
- 不能代替用户做决定——向谁说、说多少、何时说，最终由用户判断。
