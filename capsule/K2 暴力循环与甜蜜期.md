# triggers

- "为什么他总是动手对我之后又对我特别好"
- "他说会改可是下次还打我怎么办"
- "每次道歉后我都心软原谅了他"
- "他发脾气之后又会很温柔是不是没救了"
- "吵完架/动手后送东西、下跪、写保证书"

# applicability

## use_when

- 用户描述伴侣有明显的周期性“施暴-忏悔-温情-再施暴”行为模式
- 用户因对方在温情时刻（甜蜜期）的顺从表现而动摇、自责或犹豫是否离开
- 评估当前无即时人身危险（处于暴力间歇期）

## do_not_use_when

- 用户正在遭遇暴力或当前处于不安全空间
- 用户手机/财务账户正被对方监控
- 用户明确表达自伤、自残、自杀倾向
- 需要立即报警或寻求紧急避难的冲突现场

# render_policy

- 如果对方控制你的手机、通讯工具或账户，尽量使用安全设备和安全网络寻求帮助。
- 收集证据时优先考虑自身安全，不必为了取证而冒险。
- 向亲友求助前，尽量确认对方能够保密并尊重你的决定。
- 若对方情绪激动或存在暴力风险，避免当面升级冲突，优先考虑自身安全。
- 提出分手、离婚、搬离住所或采取法律措施后，部分施暴者可能出现风险升级；如有条件，可提前与专业机构讨论安全计划。

# recognize

- 暴力后的温柔、送礼、保证书或承诺可能让人重新燃起希望，但这些行为本身无法证明暴力已经结束。
- 真正的改变通常需要长期、持续、可验证的行为改变；专业干预、责任追究和外部监督往往比口头承诺更可靠。
- 长期处于暴力循环可能造成心理创伤、自我怀疑、警觉性升高、情绪困扰等影响。
- 如果暴力频率增加、伤害加重或控制行为升级，风险可能持续提高。
- 离开暴力关系通常是一个渐进过程，寻求外部支持有助于提升安全性和选择空间。

# act
<!-- steps use ordered list/choices use unordered list-->
- 记录暴力事件发生时间、经过、造成的伤害及后续情况，帮助识别模式，并在需要时保留事实资料。
- 与信任的朋友或家人分享情况，逐步建立支持网络。
- 咨询法律援助机构，了解人身安全保护令、证据保存及相关法律程序。
- 联系妇联或反家暴组织，获取安全规划、法律信息和心理支持。
- 根据自身情况考虑制定安全计划，包括重要证件、资金、紧急联系人及临时住所安排。

## scripts
<!-- 提供1到2句话术模板 列表形式，用引号包裹的句子-->
- 向妇联/法律援助机构叙述："我遭遇了长期的家庭暴力，他打完我就会道歉，过阵子又打。我想了解，如果我带孩子离开，怎么申请人身安全保护令防止他来抢孩子？"
- 向信任亲友试探求助："最近他脾气有点大，甚至跟我动了手。虽然他过后一直道歉，但我心里挺害怕的。我能先把一些重要证件寄存在你那里吗？万一有事，我也有个退路。"
