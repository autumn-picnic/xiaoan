# triggers

- 用户提到想让警察管一管/警告施暴者
- 用户提到报警后警察没有做什么/只是口头说了几句
- 用户提到想要书面证据/留下记录
- 用户提到暴力"不算严重"但想要正式处理
- 用户提到"告诫书"，不清楚“告诫书“内容
- 用户问报警之后能得到什么
- 用户问怎么让警察认真处理
- 用户已拿到告诫书，问接下来做什么

# applicability

## use_when

- 用户已经遭受家庭暴力，想了解告诫书是什么、能起什么作用
- 用户想预防施暴者再次施暴,需要正式书面警告
- 用户正在收集证据，准备离婚或申请保护令
- 用户报警后被口头调解打发，想知道自己还能争取什么
- 用户已拿到告诫书，想知道后续如何使用

## do_not_use_when

- 用户正在遭受暴力或处于即时危险中 
- 用户需要的是人身安全保护令而非告诫书 
- 施暴者不属于家庭成员或共同生活人(告诫书仅适用于《反家暴法》框架内的关系) 

# render_policy

- 告诫书是法定权利不是警察的"恩惠"，用户有权要求出具——这是本胶囊的核心立场
- 安全风险前置:在给出行动建议之前，必须提醒告诫书送达后的暴力升级风险,并确认用户有安全计划
- 不替用户判断暴力"严不严重":如果用户描述的情况可能超出告诫书适用范围，应提示可能有更强的法律工具，但不否定用户的感受
- 语气:肯定用户想留下正式记录的意识，不批评用户"为什么不早报警"
- 话术建议应从合作语气开始，逐步升级——不鼓励一上来就对抗

# recognize

- 告诫书是公安机关对实施家庭暴力的加害人出具的书面警告文件，是《反家庭暴力法》第十六条明确规定的法定制度
- 适用条件:家庭暴力情节较轻，依法不给予治安管理处罚的
- "情节较轻"的实务参考:未造成轻伤以上后果、未使用凶器、非长期反复严重暴力;但边界模糊时用户有权要求出具，由公安机关判断，不是用户自己排除自己
- 告诫书内容应包括:加害人身份信息、家庭暴力事实陈述、禁止再次施暴的警告
- 告诫书一式四份:存档一份、加害人一份、受害人一份、抄送住所地或经常居住地村/社区妇联
- 威慑功能:让施暴者知道暴力已被国家机关正式记录，再犯后果加重
- 证据功能:法院认可的家暴证据，可用于申请保护令、离婚诉讼中证明家暴事实、主张损害赔偿
- 起点功能:告诫后再次施暴，警方处理力度通常加大
- 时效性:法律未规定报警时限，但实务中越早报警越有利(现场痕迹、伤情、记忆清晰度);暴力发生数天甚至数周后仍可报警要求出具，不存在"过了时间就不能要"的规定
- 澄清常见误解:"不严重就不能报警"——错误，告诫书制度专门为"情节较轻"而设；"警察说没法管"——这是法定职责;"告诫书没什么用"——它是正式法律文书是后续维权关键证据

# act

## 分诊步骤

1. 判断用户当前处境:

- 还没报警,想了解告诫书 → 进入【准备阶段】
- 已报警/去过派出所，被口头调解或拒绝 → 进入【被拒后怎么办】
- 已拿到告诫书，问接下来做什么 → 进入【拿到之后】
- 不确定自己的情况适不适用 → 进入【适用性判断】

2. 无论哪个入口，如果用户尚未脱离施暴者控制范围,先确认安全再讨论告诫书

## 适用性判断

- 如果用户不确定自己的情况"够不够格":告诉用户告诫书适用的就是"情节较轻"的情况,不需要打到重伤才能用;法律没有要求受害人自己证明"够严重"，用户有权提出要求,由公安判断
- 如果用户描述的暴力明显严重(造成可见伤害、使用凶器、长期反复、威胁生命):告知用户其情况可能不只适用告诫书,还可以要求行政拘留甚至刑事立案,但这不妨碍同时要求出具告诫书作为记录
- 如果施暴者不是家庭成员或共同生活人:告知告诫书不适用,但可以走治安管理处罚程序(报警做笔录、要求行政处罚)

## 准备阶段

### 安全风险前置(必须先说)

- 告诫书会送达施暴者，施暴者会知道你报警并要求出具了正式文书
- 部分施暴者会因此愤怒升级，这是真实风险
- 因此在去派出所之前，先想好:告诫书出具后你住在哪里?对方能不能找到你?谁知道这件事?
- 如果担心拿到告诫书后对方报复，可以同步申请人身安全保护令 
- 不是说"有风险就不要做"，而是"做之前要有准备"

### 行动准备

- 想清楚目标:如果主要目标是预防再次暴力+留下书面证据,告诫书是合适的工具
- 携带能证明暴力的材料(如有):照片、聊天记录、就医记录、证人联系方式;没有也可以去,不是必须有证据才能要求
- 如果可能,让信任的人陪同前往
- 去辖区派出所(暴力发生地或你的居住地)

### 现场沟通策略(梯度)

- 第一步(合作):说明情况后,明确提出"我希望对他出具告诫书"
- 第二步(引用法条):如警察犹豫，补充"《反家庭暴力法》第十六条规定,情节较轻、不给予治安管理处罚的,公安机关应当出具告诫书,我正式要求出具"
- 第三步(坚持):如警察仍推脱,说"如果您认为不符合条件,请给我书面的不予出具说明,我会向上级反映"
- 注意:态度坚定但不对抗，目标是拿到文书不是和警察吵架；记下接警警察的警号和姓名

### 拿到后现场确认

- 检查告诫书是否包含三项内容:加害人身份信息、暴力事实、禁止再次施暴
- 确认自己拿到了一份(共四份,你应持有一份)
- 如有遗漏或表述不准确,当场提出

## 被拒后怎么办

- 首先确认:警察是完全拒绝出具,还是做了"调解"但没给书面文书?
- 如果是口头调解没给文书:调解不替代告诫书,你有权要求"除了调解,我还要求出具告诫书"
- 如果是明确拒绝出具:
  - 要求对方说明不出具的理由(最好书面)
  - 记录接警警察警号、姓名、处理时间
  - 拨打12389(公安机关督察电话)投诉
  - 联系当地妇联反映情况,请妇联协调(妇联有法定的协调职能)
  - 可以换一个时间、换一个值班班次再去
- 如果警察以"证据不足"为由拒绝:告诉用户告诫书的出具不要求受害人自己提供完整证据链,公安机关有调查核实的义务;但如果能补充证据(就医、照片等)再去，实务中会更顺利

## 拿到之后

- 保存:原件存放在施暴者无法接触的安全地点;拍照备份发给信任的人或存云端
- 告诫书出具后公安机关应当通知居委会/村委会跟踪回访,你可以主动联系社区确认他们知道此事
- 后续可选行动:
  - 如果担心对方不收敛 → 可同步申请人身安全保护令(告诫书本身就是保护令申请的有力证据)
  - 如果在准备离婚 → 告诫书可作为证明家暴事实的证据
  - 如果对方告诫后再次施暴 → 携带告诫书再次报警,警方处理力度通常会升级(行政拘留或刑事立案)
- 心理预期:告诫书不是万能的，有些施暴者会收敛，有些不会；它的核心价值是"留下正式记录+为后续升级处理打基础"

## scripts

按升级梯度排列:

1. (合作)"警官您好,我遭受了家庭暴力,想请您根据《反家庭暴力法》对对方出具告诫书。"
2. (坚持)"我了解告诫书适用的就是情节较轻的情况。第十六条规定公安机关应当出具告诫书,我正式提出这个要求。"
3. (施压)"如果您认为不符合出具条件,请给我书面说明理由。我会向上级公安机关反映,也会联系妇联协调。"
4. (回应"口头调解就够了")"谢谢您的调解，但口头调解不能作为证据。我需要一份书面告诫书，如果他再犯，处理起来也有依据。"
