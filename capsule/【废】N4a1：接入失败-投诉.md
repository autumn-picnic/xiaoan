# 1. 基础信息
capsule_name: "被拒后的精准投诉与证据转化"
status: "draft" 

# 2. 触发条件
triggers:
  - "居委会直接把我赶出来，说管不了家务事"
  - "妇联工作人员态度恶劣，说没有报警记录就不接待"
  - "打12345反映社区不管家暴，热线只是记录了但一直没动静"
  - "被拒绝了，我很气愤，想让他们付出代价或者留下他们不作为的证据"

# 3. applicability 适用规则
use_when:
  - "用户向居委会、妇联、街道办求助被明确拒绝、推诿或冷暴力对待"
  - "用户目前处于相对安全的物理环境（如已撤离或施暴者不在场），有精力进行维权操作"
  - "用户具有长线诉讼思维，希望将基层的“不作为”转化为日后离婚或申请保护令的筹码"

do_not_use_when:
  - "用户处于危险中，施暴者正试图通过监控其手机、行踪来实施报复"
  - "用户投诉的目的是为了泄愤或要求立刻“抓捕施暴者”（居委会无此权力，需做普法拦截）"

# 3. recognize 认知引导
recognize.purpose: "扭转用户的受挫心理。将‘被拒绝的绝望’转化为‘收集行政不作为证据的兴奋感’。明确投诉的终极目的：不是为了让办事员停职，而是为了向法官证明‘我已经穷尽了一切社会救助手段，受害者已尽最大努力自救，夫妻感情确已完全破裂’。"

content:
  - "妇联是群团组织，居委会是自治组织，它们不是行政机关。不能去检察院告它们‘行政不作为’，这条路走不通。"
  - "每一次被拒绝，只要你留下了录音、接访回执、甚至邮寄信件的存单，都是在丰富你未来的法庭证据链。"
  - "基层组织最怕‘12345工单挂账’和‘上级主管部门督办’，利用行政KPI逼他们低头。"

# 4. act 行动指引
act.purpose: "指导用户避开维权雷区，通过合法的上下级行政逻辑和12345工单机制，高效进行精准投诉，并在过程中闭环固化诉讼证据。"

steps:
  - "**第一步：现场证据固化（合法录音）**"
    "在公共场所或本人参与的对话中录音录像完全合法。确保录音中包含：求助时间、办事大厅背景音、工作人员推诿的原话（如：‘这是家务事我们管不了’）。"
  - "**第二步：退而求其次索要凭证**"
    "不要强求对方开具不存在的‘不予受理通知书’。而是要求在你的《求助登记表》上签字表明‘已阅’，或者要求给出‘接访回执’。拒绝给回执的，直接拍照留存办事大厅画面。"
  - "**第三步：EMS 邮寄‘法定时效启动’**"
    "若现场被轰走，写一份《家暴求助及申请社区介入函》，通过 EMS 邮寄给居委会主任和街道办负责人，快递面单写明内件名称。**快递签收件即成为‘官方知情且拒绝履职’的铁证**。"
  - "**第四步：利用 12345 KPI 考核精准投诉**"
    "拨打 12345 投诉该社区。话术必须精准定性：定性为基层组织违反《反家暴法》第十三条，拒绝履行法定救助协助职责，导致受害者处于持续人身危险中，请求区民政局和街道办立刻督办。"

scripts:
  - "12345 投诉话术：“我要投诉 XX 社区居委会。我于 X 月 X 日前往该社区反映家暴并求助，工作人员违反《反家庭暴力法》第十三条，以‘家庭纠纷’为由推诿、拒绝履行法定协助职责。由于社区的不作为，导致我目前处于持续的人身危险中。我请求其上级主管单位（区民政局/街道办事处）立刻督办，并书面回复我处理结果，该结果我将作为日后离婚诉讼的证据提交法院。”"
  - "向上一级妇联投诉话术：“我向 XX 街道妇联反映家暴求助，工作人员以‘无出警记录不予立案’为由拒绝提供任何维权协助。这严重违反了妇联组织维护妇女权益的章程。我请求上级妇联权益部介入督办，并责令其纠正错误工作流程。”"

safety_note:
  - "**致命评估（红线）**：如果施暴者在本地有广泛人脉，或者与居委会工作人员熟识，**严禁盲目投诉**！投诉工单下发到社区可能会变成‘通风报信’，引致极其暴烈的报复。此种情况必须优先撤离，绝不惊动基层。"
  - "备份防删：所有投诉电话的录音、12345的短信回复截图，必须立刻同步到隐藏云端或加密U盘，防止手机被施暴者抢夺砸毁。"

# 5. ground 依据与限制
legal_basis:
  - "《中华人民共和国反家庭暴力法》第十三条：有关单位接到投诉、反映或者求助后，应当给予帮助、处理。"
  - "《中华人民共和国妇女权益保障法》：各级妇联组织应当维护妇女合法权益，对侵害妇女权益的行为有权要求有关部门查处。"
limits:
  - "投诉属于行政考核和内部督办流程，整改和回复通常需要 3-7 个工作日，**无法解决当下的刀砍斧凿**。如遇紧急危险，本胶囊完全失效，请立刻报警或逃生。"
