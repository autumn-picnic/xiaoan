# **"报警了但警察说是家务事不管，怎么办？"** 

> **核心原则**：警察说“这是家务事”是**错误的执法惯性**，绝非法律定论。你的每一次坚持，都是在推动执法规范化。 **适用场景**：遭遇家暴报警后，警方出警迟缓、定性为“家庭纠纷”、拒绝出具笔录或告诫书、拒绝立案。




##  Recognize / 认知：打破“家务事”迷思，确认你的权利

### 1. “家务事”是过时的执法借口，不是法律事实

- **误区**：很多人认为“清官难断家务事”，警察来了只能“劝和”。
- **真相**：自2016年《反家庭暴力法》实施以来，**家庭暴力是违法行为，不是家务事**。2025年修订的《治安管理处罚法》及2026年最新执法规范明确指出：**公安机关必须出警、制止、调查取证**。

### 2. 你的恐惧和犹豫是正常的，但沉默会让暴力升级

- **心理陷阱**：受害者常因“怕把事情闹大”、“怕警察不管”、“怕被说小题大做”而放弃报警。
- **现实风险**：多项研究和实务经验表明，若首次家暴未受到有效法律干预，复发率极高（通常估算在60%-80%），极易升级为严重伤害。
- **你的权利**：你不需要“完美受害者”的身份才配报警。只要发生暴力，你就有权要求警方**依法履职**。

### 3. 警察的“推诿”可能是程序疏忽，也可能是失职

- **常见话术**：“你们夫妻吵架我们管不了”、“回去好好谈谈”、“没打伤就不算事”。
- **法律定性**：这些回应违反了《反家庭暴力法》第15条和《治安管理处罚法》第73条。**拒绝出警、拒绝记录、拒绝告诫**属于**行政不作为**，你可以投诉。




## Act / 行动：低负担、高策略的“四步反击法”

### ✅ 第一步：报警时的“黄金话术”（现场控制）

**目标**：让警方无法用“家务事”搪塞，强制启动执法程序。

- **话术模板**（请大声、清晰地说）：“我是**受害人**，我**正在遭受家庭暴力**（或刚刚遭受），对方**殴打/威胁**我，我的人身安全受到**现实危险**。我要求**立即出警**，**制止暴力**，并**依法调查取证**！”
    - ❌ **避免说**：“我们吵架了”、“他脾气不好”、“家里有点矛盾”。（这些会被定性为纠纷）
    - ✅ **必须说**：“**被打**”、“**暴力**”、“**人身安全受威胁**”。
- **关键动作**：
    1. **开启录音/录像**：报警全程录音，记录警察的回应。
    2. **要求出警记录**：警察到场后，明确要求：“请**制作出警笔录**，记录我的伤情和施暴事实，不要只写‘家务纠纷’。”
    3. **要求伤情鉴定**：如果有伤，当场要求：“请带我去医院验伤，或协助我进行**法医伤情鉴定**。”

### ✅ 第二步：现场取证与文书固定（证据固化）

**目标**：将“口头纠纷”转化为“法律案件”。

- **必须索要的文书**：
    1. **《接处警记录》**：必须包含时间、地点、当事人、简要案情。
    2. **《家庭暴力告诫书》**（核心武器）：
        - **依据**：《反家庭暴力法》第16条。
        - **话术**：“根据《反家庭暴力法》第16条，情节较轻的也应出具**告诫书**。请依法开具，并送达给我和居委会。”
        - **作用**：告诫书是法院认定家暴的**铁证**，也是后续申请保护令的关键。
    3. **现场照片**：要求民警拍摄现场、伤情、破损物品。
- **如果警察拒绝出具告诫书**：
    - **话术**：“如果您认为不需要出具，请**在笔录中写明理由**，并告知我法律依据。否则我将视为**行政不作为**。”
    - **行动**：当场拨打**12389**（公安机关督察电话）或**12345**（市民热线）进行投诉，并录音。

### ✅ 第三步：事后补救与投诉（强力维权）

**目标**：纠正警方错误，留下追责记录。

- **投诉渠道**（按优先级排序）：
    1. **12389**（公安机关违纪违法举报平台）：最直接，针对警察不作为。
    2. **12345**（市长热线）：转办至政法委或公安局督察。
    3. **检察院监督**：向当地**人民检察院**申请**立案监督**或**行政违法行为监督**（2026年检察公益诉讼已覆盖家暴领域）。
    4. **妇联协助**：联系当地妇联（12338），请求其向公安机关发出\*\*《支持起诉意见书》**或**《法律监督建议书》\*\*。
- **投诉话术**：“我于X月X日X时在XX地报警遭家暴，出警民警以‘家务事’为由拒绝出警/拒绝制作笔录/拒绝出具告诫书，违反《反家庭暴力法》第15条。我要求**督察部门介入调查**，并**责令其依法履职**。”

### ✅ 第四步：升级法律程序（保护令与诉讼）

**目标**：绕过警方推诿，直接寻求司法保护。

- **直接申请人身安全保护令**：
    - **依据**：《反家庭暴力法》第23条。
    - **操作**：即使警方未出具告诫书，只要有**报警记录（哪怕只是接警单）**、**伤情照片**、**录音**，即可向法院申请。
    - **话术**：“虽然警方未出具告诫书，但我有出警记录和伤情，证明存在现实危险，请法院依法受理。”
- **提起行政诉讼**：
    - 如果警方明确拒绝出警或拒绝履行职责，可向法院提起**行政诉讼**，告公安机关“行政不作为”。




## Ground / 依据：法律、规范与实务边界

| 法律文件 | 关键条款 | 你的权利 |
| --- | --- | --- |
| **《反家庭暴力法》** | **第15条**：公安机关接到报案后应当**及时出警**，制止暴力，调查取证，协助就医、鉴定。 | **强制出警权**：警察无权以“家务事”拒绝出警。 |
|  | **第16条**：情节较轻不处罚的，应出具**告诫书**。 | **告诫书请求权**：即使不打人，只要有家暴行为，就应出具。 |
|  | **第35条**：未报案造成严重后果的，责任人受处分。 | **追责权**：警察不报案/不处理可被处分。 |
| **《治安管理处罚法》** | **第73条**（2025修订）：拒不执行禁止家暴告诫书的，处拘留、罚款。 | **处罚依据**：违反告诫书可直接拘留。 |
|  | **第51条**：殴打他人，处拘留、罚款；殴打孕妇、老人、儿童从重。 | **处罚标准**：家暴不是“轻微”，而是“从重”。 |
| **《妇女权益保障法》** | **第65条**：有关部门应预防和制止家暴，提供救助。 | **多部门联动**：妇联、公安、法院必须联动。 |

### 

### ⚠️ 边界与局限（Reality Check）

1. **证据门槛**：虽然法律要求出警，但如果**没有任何证据**（无伤、无录音、无证人），警方可能难以立案。
    - *对策*：**即使没伤，也要坚持要求出警和记录**，因为“未造成伤害”不代表“没有暴力”，记录本身就是证据。
2. **地域差异**：一线城市执法规范，偏远地区可能仍存在“和稀泥”。
    - *对策*：遇到推诿，**立即投诉**（12389），并同步联系**检察院**和**妇联**介入。
3. **调解陷阱**：警方可能建议“调解”。
    - *对策*：**家暴案件原则上不适用调解**。如果必须调解，必须在**安全、自愿**前提下，且**不能免除施暴者法律责任**。
4. **时间成本**：投诉和诉讼需要时间。
    - *对策*：**同步进行**：一边投诉警方，一边向法院申请**保护令**，不要等投诉结果。




## 锦囊1：给不同场景的特别建议

- **给“第一次报警”的你**：
    - **不要怕**：第一次报警是**最关键**的。警方态度越严厉，复发率越低。
    - **不要妥协**：拒绝“私下和解”，坚持**出具告诫书**。
- **给“多次报警无效”的你**：
    - **升级投诉**：从派出所升级到**分局督察**，再到**市检察院**。
    - **媒体曝光**：在确保安全前提下，联系正规媒体或公益组织（如“反家暴联盟”）。
- **给“担心报复”的你**：
    - **匿名报警**：如果担心身份泄露，可要求**匿名报警**（但需配合调查）。
    - **保护令先行**：先申请保护令，再报警，让警方知道你有法律后盾。
