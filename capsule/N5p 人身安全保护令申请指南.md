# applicability 适用规则
use_when:
- 用户正在经历或担忧对方的跟踪、骚扰、威胁、接近行为
- 用户有意愿通过法律途径建立物理安全边界
- 用户想了解保护令的申请条件、流程、证据要求
- 用户已有保护令但对方违反，需要知道下一步怎么做
- 用户遭受精神暴力（谩骂、恐吓、侮辱、跟踪等）但不确定是否"够格"申请
- 用户对保护令存在误解需要纠偏

do_not_use_when:
- 用户正在遭遇暴力或处于紧急人身危险中（应先转至紧急求助模块）
- 用户有自伤风险或表达活不下去（应先转至危机干预模块）
- 用户描述的情况不属于家庭成员或共同生活关系之间的暴力（不适用反家暴法）

# recognize 认知引导
recognize.purpose: "让用户理解保护令是一个真实可用、门槛不高的法律工具——不需要先离婚才能申请，不需要等到被打了才能申请，不需要有明显外伤，精神暴力同样受保护。"
content
- 面临家庭暴力的现实危险就可以申请，不需要等到被打了
- 申请保护令不以提起离婚等民事诉讼为条件，可以完全独立申请
- 保护范围包括家庭成员以及同居情侣、儿媳/女婿/公婆等共同生活的人
- 法院审查标准为"较大可能性"——无需达到严格诉讼标准，能有多少证据算多少
- 完全免费，不收取任何诉讼费用
- 一般情况72小时内作出裁定，紧急情况24小时内出具
- 违反保护令可处罚款、司法拘留（最高15天），情节严重追究刑责
- 常见误解：先离婚才能申请（错）；没有严重伤情不能申请（错）；保护令只是废纸（错）

# act 行动指引
act.purpose: "将'申请保护令'拆解为清晰的操作流程，让用户知道去哪里、带什么、怎么写、提交后会发生什么。"

steps:
1. 确认管辖法院（免费）：向以下任一基层法院申请——申请人居住地、被申请人居住地、家庭暴力发生地
2. 准备身份与关系证明（必备）：申请人与被申请人的身份证；证明双方关系的材料（结婚证、户口本、同居证明等）
3. 收集家暴证据（能有多少算多少）：报警回执/告诫书/行政处罚决定书；照片（伤痕、打砸现场）；病历资料；录音/录像/短信/聊天记录；证人证言；妇联/社区求助记录；对方保证书/悔过书。无法自行收集的，可申请法院调查收集
4. 撰写申请书：写明①双方基本信息；②具体请求（禁止暴力/骚扰/接触/迁出住所等）；③事实及理由（具体暴力经过）。书写困难可到法院口头申请，由工作人员记录成笔录
5. 提交申请：到法院立案庭窗口递交，或委托近亲属/妇联/居委会等代为申请，部分法院支持线上递交
6. 等待裁定与执行：72小时内（紧急情况24小时内）作出裁定；裁定后送达双方并抄送辖区派出所和居委会/村委会；有效期最长6个月，到期前可申请延长
7. 对方违反保护令时：立即报警并告知你持有保护令；记录违反事实（拍照、录像、保留通话记录）；违反者将面临罚款、拘留或刑事追责

申请书模板框架：
  申请人：姓名、性别、出生日期、住址、联系方式
  被申请人：姓名、性别、出生日期、住址、联系方式
  申请人与被申请人关系：（夫妻/同居/亲子/其他）
  申请请求：1.禁止被申请人对申请人实施家庭暴力；2.禁止骚扰跟踪接触；3.责令迁出住所（根据需要增减）
  事实与理由：（按时间顺序写明每次暴力的时间、地点、方式、后果及报警/就医/求助情况）
  此致 ______人民法院

safety_note
- 申请保护令可能激怒对方，提交前先评估当前居住安全，必要时同步准备安全离开计划
- 如果对方监控手机，不要在手机上搜索保护令相关内容，可通过妇联或律师代为递交
- 保护令不是物理隔离——裁定后如果对方仍然出现，立即报警并记录违反事实
- 就医时在病历中明确写出：受伤时间、地点、经过、加害人姓名和关系

- 法发〔2022〕10号：最高法等七部门《关于加强人身安全保护令制度贯彻实施的意见》：规定各部门协同执行职责；法院作出保护令后24小时内送达公安、居委/村委
- 公通字〔2024〕34号：九部门《关于加强家庭暴力告诫制度贯彻实施的意见》：细化告诫书出具标准及联动机制

practice_basis
- 上海五年数据（2016—2020，179起样本）：总体核发率51%；提交两项及以上证据的申请核发率达73%；有告诫书的案例核发率约90%；证人证言采纳率100%
- 核准率最高的措施：禁止实施家庭暴力（76%）；禁止骚扰跟踪接触（50%）；责令迁出住所仅16%
- 处理时限：77%在72小时内完成；23%超时，平均处理时间4天，最长29天
- 有效期：86%的保护令有效期为6个月；93%以上自动履行，未发现需强制执行案例
- 特殊群体：未成年人申请核发率78%，老年人约74%，均高于总体平均值
- 证据效力排序（实务）：证人证言（100%）> 告诫书（约90%）> 音视频/就医材料/相关部门证明（>80%）；施暴者道歉保证书被驳回比例>40%，不宜单独作为主要证据
- 常见违法拒受理理由（应向上级法院或检察机关投诉）：要求至少两份报案回执、下午不受理、需与离婚诉讼合并处理
- 常见违法驳回理由：申请人与被申请人不同住（不符合法律）；被申请人认错道歉（不应作为驳回理由）；暴力频率低或非近期（保护令针对"面临现实危险"，不以频繁发生为前提）；双方无其余房产（不能作为驳回迁出令的理由）
- 离婚期间申请核发率（43%）低于总体水平（51%），需提前准备
- 数据来源：为平妇女权益机构《上海市人身安全保护令实施情况研究报告》（2021年3月）

limits
- 不能保证法院一定签发保护令，法官有裁量权，但证明标准为"较大可能性"
- 保护令有效期最长6个月，到期需申请延长
- 不能承诺对方一定遵守——保护令是法律威慑，不是物理隔离
- 不能替代报警——如果对方正在实施暴力，应先报警
- 各地保护令申请方式、证据把握尺度、执行力度因地区而异，建议通过12368或当地妇联确认本地流程

# route 路由

route.purpose: 保护令是强有力的法律工具，需要引导用户了解申请条件、准备材料，以及与其他维权行动的协同。

## 前置准备

| 用户情况 | 路由去向 | 触发判断 |
|---------|---------|---------|
| 不确定是否够严重 | 保持本胶囊 | 使用recognize板块打破"不够格"误解 |
| 没有证据 | → 证据收集指南 / → N3a 报警 | 用户说"什么证据都没有" |
| 想先报警 | → N3a 报警 | 报警记录可作为申请保护令的证据 |

## 并行路由（建议同步进行）

| 用户情况 | 路由去向 | 触发判断 |
|---------|---------|---------|
| 需要安全撤离计划 | → N2 安全准备 | 申请保护令可能激怒对方 |
| 考虑离婚 | → N5d 是否离婚 / → N5e 离婚诉讼求助 | 保护令可独立申请，也可在离婚诉讼中一并提出 |
| 需要社区支持 | → N3c 亲友·社区·机构 | 妇联、居委会可协助申请或代为申请 |

## 后续路由（获得保护令后）

| 用户情况 | 路由去向 | 触发判断 |
|---------|---------|---------|
| 对方违反保护令 | → N3a 报警 | 立即报警，记录违反事实 |
| 保护令即将到期 | 保持本胶囊 | 引导申请延长 |
| 考虑离婚 | → N5d 是否离婚 / → N5e 离婚诉讼求助 | 有保护令后离婚更安全 |
| 需要心理支持 | → 心理支持 | 申请过程或被拒后的情绪支持 |

## 特殊路由（申请被驳回时）

| 用户情况 | 路由去向 | 触发判断 |
|---------|---------|---------|
| 第一次被驳回 | 保持本胶囊 | 引导补充证据后再次申请 |
| 需要投诉或上诉 | → 法律援助 | 寻求专业律师协助 |
| 绕过保护令，直接离婚 | → N5e 离婚诉讼求助 | 在离婚诉讼中再次主张保护 |
