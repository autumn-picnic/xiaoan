# N6a：分开后持续骚扰



## 1. 基础信息

**capsule_name:** "分开后持续骚扰"

**status:** "draft_v0"



## 2. 触发条件

```yaml
triggers:
  - "前男友一直发骚扰短信威胁我"
  - "分居后对方总是来我家楼下堵我"
  - "怕前夫知道我在哪，一直在跟踪我"
  - "离婚了但他还不停打电话骂我"
```



## 3. 适用规则

```yaml
use_when:
  - "用户已确认与施暴者分开（分居、离婚或结束恋爱关系）"
  - "用户当前报告的是骚扰、跟踪、精神恐吓等非即时肢体暴力行为"
  - "用户有寻求法律或社区干预的意愿，且评估暂无迫在眉睫的生命危险"

  fits_any:
  - "用户已确认与施暴者分开（分居、离婚或结束恋爱关系）"
  - "用户当前报告的是骚扰、跟踪、精神恐吓等非即时肢体暴力行为"
  - "用户有寻求法律或社区干预的意愿，且评估暂无迫在眉睫的生命危险"
```

```yaml
do_not_use_when:
  - "用户正在遭遇暴力或处于紧急人身危险中（应先转至紧急求助模块）"
  - "用户明确表示无法安全前往法院且无法让他人代为申请，且当前未处于危险中（应先处理安全计划）"
  - "用户有自伤风险或表达活不下去（应先转至危机干预模块）"
  - "用户描述的情况不属于家庭成员或共同生活关系之间的暴力（不适用反家暴法）"
```



## 4. 认知引导（recognize）

**recognize.purpose:**
"帮助用户识别骚扰行为的违法本质，确认其恐惧的合理性，并建立‘我有权利不被打扰’的安全感。"

```yaml
content:
  - "骚扰和跟踪是家庭暴力的延续，也是违法行为，不是你的错。"
  - "分开不代表关系彻底终结，法律保护你免受后续侵害。"
  - "不要试图通过讲道理感化施暴者，这往往会激化矛盾。"
  - "所有的骚扰记录都是将来申请保护令或起诉的重要证据。"
  - "你的安全和心理安宁是第一位的，不需要为了“体面”而忍受。"
```


## 5. 行动指引（act）

**act.purpose:**
"引导用户从被动忍受转向主动取证和法律维权，将不可控的焦虑转化为具体的防御步骤。"

```yaml
steps:
  - "立即停止一切形式的回应：不回复消息、不接电话、不与其见面，不给对方任何情绪反馈。"
  - "系统收集证据：对每一条骚扰短信、微信、邮件进行截图（保留时间戳），对通话记录进行录屏或录音，详细记录每一次被跟踪的时间、地点、对方行为特征（可手写日记）。"
  - "向公安机关报案并索取回执：明确告知警察“我正遭遇骚扰和跟踪”，要求警方出具《接处警记录》或《家庭暴力告诫书》，这是关键的法律文书。"
  - "向人民法院申请人身安全保护令：依据《反家庭暴力法》第三十七条，即使已非共同生活，骚扰行为仍可参照执行；申请禁止对方骚扰、跟踪、接触你及其近亲属，并划定禁入区域。"
  - "通知关键人员：将情况告知家人、朋友、同事、房东或学校老师，请求他们协助留意异常情况，并提供必要的证词。"
```

```yaml
scripts:
  - scene: "面对警察陈述"
    say: "我已于X月X日与对方分开，但现在他持续对我进行骚扰、跟踪和威胁，我已经没有安全感，请警方帮我出具告诫书。"

  - scene: "向法院申请保护"
    say: "本人申请人身安全保护令，请求禁止被申请人继续通过电话、短信骚扰我，禁止其在我住所及工作地点附近出现。"
    
```

```yaml
safety_note:
  - "如果对方掌握你的行踪（如监控你的手机），请在公共场所或有信任的人陪同下操作报警或取证。"
  - "不要单独去见对方，也不要轻易透露新的居住地址或工作单位。"
  - "在获得人身安全保护令前，避免独自深夜出行。"
```



## 6. 依据与限制（ground）

**ground.purpose:**
"明确法律依据，强调人身安全保护令的强制力及对违反行为的惩戒措施，管理用户对‘劝和’的不切实际期望。"

legal_basis:
  - "《中华人民共和国反家庭暴力法》第二条、第三十七条（家庭成员以外共同生活及骚扰认定）"
  - "《中华人民共和国反家庭暴力法》第二十九条、第三十四条（保护令内容及违反后果：罚款、拘留）"
