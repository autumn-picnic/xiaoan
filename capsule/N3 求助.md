# triggers
- "我该怎么办"
- "我不知道从哪里开始"
- "我需要帮助"
- "我想求助"

# applicability
use_when:
- 用户首次求助、不知从何开始
- 用户想了解可以采取哪些行动
- 用户需要被引导聚焦到具体方向

do_not_use_when:
- 用户正在遭遇暴力或当前不安全 → 转【即时危机】
- 用户已明确知道自己需要什么（如直接询问报警、法律等具体问题）→ 转对应胶囊

# recognize

1. **愿意开口求助本身已经是重要的一步。** 很多人在这一步之前已经独自承受了很久。

2. **报警不是第一步，也不是唯一选项。** 如果马上报警是唯一被给到的选项，可能反而加剧暴力危险。情绪缓和、距离隔开之后，110介入才更有效、更安全。

3. **先拉开距离、确保安全，再逐步收集支持，最后才是法律行动。** 顺序可以根据处境灵活调整，任何情况下答案都不是唯一的。

4. **有人陪伴会让后续每一步都更好走。** 打破孤立本身就是行动。

# act

## 首先确认安全
- 你现在安全吗？他在你身边吗？
- 如果当前不安全，优先转移到安全环境（亲友家、派出所、庇护机构）

## 求助参考顺序（非固定，根据处境调整）

| 优先级 | 行动 | 核心价值 | 备注 |
|--------|------|----------|------|
| 1 | **亲友支持** | 打破孤立，获得陪伴和临时安置 | 即使只是一个人知道你的情况，也是保护 |
| 2 | **躲避/拉开距离** | 让自己脱离暴力现场 | 派出所也可以作为临时躲避场所，不一定要"报案"才能去 |
| 3 | **就医/120** | 治疗伤情 + 就医记录即证据 | 120医护人员一定会到场；先到医院也能间接获得帮助 |
| 4 | **报警/110** | 制止暴力 + 法律记录 | 更适合在情绪缓和、距离隔开后介入；有生命健康危险或已出现伤口时才需立即拨打 |
| 5 | **法律救济** | 告诫书/保护令/离婚 | 需要准备，不急于此刻 |

**核心原则：** 先拉开距离确保安全 → 再获得陪伴支持 → 再法律行动。顺序随时可调整。

## 关于报警的补充说明
- 有生命健康危险或已经出现伤口 → 立即拨打110或120
- 暴力正在发生但尚未造成严重伤害 → 优先离开现场，到安全地方后再报警
- 120可能比110更可靠：医护人员一定会到场，警察可能不到场
- 派出所不仅是报警点，也可以是临时躲避场所（直接走进去即可）
- 事后报警（去派出所做笔录）同样有效，不一定要"当场"

## 引导话术
- "你愿意说出来，已经迈出了很重要的一步。你现在最想了解哪方面——怎么让身边人帮你、怎么先离开危险环境，还是法律上怎么保护自己？"

# route 路由

route.purpose: 作为求助总览与分流胶囊，根据用户当前处境和最迫切需求，引导至相应的具体行动胶囊。

## 即时安全评估

| 用户情况 | 路由去向 | 触发判断 |
|---------|---------|---------|
| 正在遭遇暴力或当前不安全 | → 即时危机处理 | 用户说"他在我身边""正在打我""现在很危险" |
| 有生命健康危险 | → 立即拨打110或120 | 用户说"有刀""威胁杀我""严重受伤" |

## 主要分流方向

| 用户情况 | 路由去向 | 触发判断 |
|---------|---------|---------|
| 想报警或刚发生暴力 | → N3a 报警 | 用户问"怎么报警""要不要报警""刚被打了" |
| 需要人际支持或机构帮助 | → N3c 亲友·社区·机构 | 用户说"不知道跟谁说""有什么机构""需要找人帮忙" |
| 想提前准备或计划离开 | → N2 安全准备 | 用户说"想提前准备""怎么离开""担心他报复" |
| 需要法律保护 | → N5p 人身安全保护令 | 用户问"怎么保护自己""法律能帮我吗""怕他骚扰" |

## 后续可能路由

| 用户情况 | 路由去向 | 触发判断 |
|---------|---------|---------|
| 考虑是否离婚 | → N5d 是否离婚 | 用户问"该不该离婚""离婚流程" |
| 需要情绪支持 | → 心理支持 | 用户表达崩溃、无助、抑郁情绪 |
| 需要证据指导 | → 证据收集与法律准备 | 用户问"需要什么证据""怎么收集证据" |

# ground

## 法律依据
- 《中华人民共和国反家庭暴力法》（2016）

## 实践依据
- 一线社工经验：报警作为唯一选项可能加剧暴力危险程度；情绪缓和+距离隔开后110介入更有效（工作坊讨论）
- 120医护人员到场率高于警察出警率（一线反馈）
- 派出所可作为临时躲避场所使用

## 局限
- 不能替代专业社工或法律人员的评估
- 求助顺序仅供参考，不适用于所有处境
- 本胶囊为概览与分流，具体行动指导由对应胶囊提供
