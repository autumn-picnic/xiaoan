## 4. 认知引导（recognize）

**recognize.purpose:**
"打破'说出来=丢人/软弱/会更糟'的心理障碍，帮助用户理解：让一个人知道你的处境，本身就是在为自己建立安全网。同时帮助用户理解支持网络的战略意义——拥有一个支持网络往往比单次法律行动更能持续保护你。"

**content:**
```
content:

  # ———— 为什么要说出来 ————

  - theme: "说出来的意义"
    points:
      - "把正在经历的事情告诉一个人，不是软弱，不是'家丑外扬'，
         而是在为自己建立一个支撑点。
         施暴者最怕的就是你不再孤立。"

      - "你不需要等到'够严重'才可以开口——
         觉得不对劲，就已经够了。"

      - "单次的法律行动可以阻止一次暴力，
         但只有支持网络的建立才能改变你在关系中的结构性弱势。
         拥有一个哪怕只有两三个人的支持网络，
         往往比报警更能长期帮助到你。"

  # ———— 选人的原则 ————

  - theme: "怎么选对人"
    points:
      - "选人需要策略性评估——
         施暴者通常在外人面前表现良好，你跟朋友讲了可能不被相信；
         消息如果传到施暴者耳朵里反而会激化危险；
         有些'支持者'可能本身带有偏见，给出的建议反而是二次伤害。
         所以选人不是凭感觉，是需要仔细评估的。"

      - "安全性比亲疏远近更重要：
         这个人会不会转头告诉施暴者？
         如果你不确定，先不要选这个人。"

      - "能听你说、不急着评价你，比能'帮你解决问题'更重要。
         你现在需要的是被听见和被相信。"

  # ———— 说不好/被拒怎么办 ————

  - theme: "开口的心理准备"
    points:
      - "一个人的不良反应不代表所有人都会这样——
         如果第一个人没接住你，你有权利试第二个人。"

      - "你不需要一次讲清所有事情，也不需要'表述完美'。
         说到一半哭了、说得颠三倒四、说完觉得'好像也没那么严重'——
         这些都很正常。"

  # ———— 如果身边真的没人 ————

  - theme: "没有人可以说时"
    points:
      - "如果身边确实暂时没有可以安全开口的人，这不是你的问题——
         还有专业机构可以联系。（→ N3c-2）"
```

## 5. 行动指引（act）

**act.purpose:**
"帮助用户系统评估身边谁是安全的、谁是可信的、谁有能力帮上忙、谁可能帮倒忙、以什么方式告知风险最低；然后将'向别人开口'这件心理门槛极高的事拆解为具体可操作的小步骤。"

**step_0_evaluate:**
```
step_0_evaluate:
  purpose: "在开口之前，先做一个系统性的人际安全评估"
  questions:
    - "这个人和施暴者有没有交集？（共同朋友、同事、亲属）"
    - "这个人以前知不知道施暴者对外表现的'好形象'？会不会因此不信你？"
    - "这个人能不能保密？以前有没有泄露过别人的秘密？"
    - "这个人的价值观怎样？会不会说'一个巴掌拍不响'或'为了孩子忍忍'？"
    - "这个人有没有实际帮助的能力？（哪怕只是一个沙发让你暂住）"
    - "告诉这个人之后，如果消息传到施暴者那里，后果有多严重？"
  categories:
    - category: "优先选择"
      examples: "和你伴侣没什么交集的朋友、你这边家人中比较开明的那个、你信任的同事、很久没联系但你知道靠得住的老朋友"
    - category: "暂时排除"
      examples: "和施暴者关系也近的人、过去劝你'忍一忍'的人、嘴快爱传话的人、容易冲动会'替你出头去找对方理论'的人"
    - category: "需要谨慎评估"
      examples: "父母（可能心疼你但也可能劝和）、共同朋友（可能两边传话）、邻居（有能力帮忙但也可能传闲话）"
```

**steps:**
```
steps:
  - step: "想一想谁可能合适"
    detail: "用上面的评估框架过一遍你认识的人。不需要找到'完美'的人——只要满足'不会告诉施暴者'和'不会反过来责怪你'这两条，就可以考虑。列出2-3个候选人。"
  - step: "选择开口方式"
    detail: "不一定要当面说。发文字消息也完全可以——有些话写下来反而更容易说出口。也可以打电话。选你觉得最没有压力的方式。"
  - step: "用一句话试探"
    detail: "不需要一开始就把所有事倒出来。可以先试探对方的反应：'我最近家里有些事情让我很不安，能不能跟你说说？'或者'我跟你说一件事，你先别急着给建议，我就是需要有人听。'"
  - step: "说出来"
    detail: "可以说多少就说多少。'我不确定我经历的算不算家暴，但我想说出来'——这就够了。你不需要为'表述不完美'道歉。"
  - step: "告诉对方一个具体请求"
    detail: "对方可能问'我能帮你做什么'。你可以提一个小而具体的要求：'帮我保密就好'、'如果我三天没回你消息你主动联系我'、'帮我保存几张截图我怕被删'、'如果我突然说要过来你能让我暂住吗'。不必一次提完所有要求。"
  - step: "建立紧急暗号（进阶）"
    detail: "和信任的亲友或邻居提前约定暗号——比如：当对方微信收到两个特定表情符号时，需要代为报警；或者你发一条特定短信，对方就知道你需要立刻被接走。暗号要简单、不引起施暴者怀疑。邻居可约定：家暴发生时以'借东西'为由敲门，有可能打断暴力。"
```

**if_response_is_bad:**
```
if_response_is_bad:
  - situation: "对方说了让你难受的话（'他平时看着挺好的啊''你是不是也有问题''要不再给一次机会''别闹大了'）"
    guidance: "这只意味着这个人不是合适的支持者，不代表你不该说出来，也不代表所有人都会这样回应。你可以说'谢谢你听我说，我需要再想想'，结束对话，不必争论。然后试第二个人。"
  - situation: "对方表示想帮忙但要'去找他理论'"
    guidance: "直接告诉对方：'请你什么都不要做，我现在需要的就是你知道这件事。'亲友善意的对质可能激怒施暴者，让你处境更危险。"
  - situation: "对方说'你这种情况很普遍'或'这是你的家务事'"
    guidance: "你可以回应：'普遍不等于正确，而且这不是家务事，这是违法行为。'但如果对方显然不理解，不必争论，换人。"
  - situation: "对方说'想想你的孩子'"
    guidance: "你可以说：'我正是为了让孩子有一个没有暴力的家才在想办法。'但同样，如果对方固执，换人比说服更重要。"
  - situation: "身边确实没有人可以安全地说"
    guidance: "这很常见，不是你的问题。专业机构每天接到的求助者里很多都是独自打来的——→转N3c-2"
```

## 6. 依据与限制（ground）

**ground.purpose:**
"提供社会支持对家暴受害者保护作用的依据，以及亲友可以承担的法律角色，同时诚实说明人际求助的局限性。"

**legal_basis:**
```
legal_basis:
  - "《中华人民共和国反家庭暴力法》第13条：家庭暴力受害人及其法定代理人、近亲属可以向加害人或者受害人所在单位、居民委员会、村民委员会、妇女联合会等单位投诉、反映或者求助。"
  - "《中华人民共和国反家庭暴力法》第23条：当事人因受到强制、威吓等原因无法申请人身安全保护令的，其近亲属可以代为申请——即亲友不只是情感支持角色，必要时可代为行使法律保护申请权。"
  - "《关于办理人身安全保护令案件适用法律若干问题的规定》进一步明确：近亲属、公安机关、民政部门、妇女联合会、居民委员会、村民委员会、残疾人联合会、依法设立的老年人组织、救助管理机构等均可代为申请保护令。"
```



**practice_basis:**
```
practice_basis:（需要修改）

-工作坊
-疫情期间反家暴指南
-
  - "家暴研究一致表明，社会孤立是家暴持续和升级的关键风险因素之一；打破孤立（即让至少一个外部的人知晓情况）本身就具有保护性作用。"
  - "实务中，受害者的亲友日后可作为证人，证明家暴事实的存在（如受害者曾向其诉说遭遇、展示伤痕等）。"
  - "受害者首次披露遭遇时获得的反应（被相信 vs. 被质疑）会显著影响其后续求助行为——因此本胶囊强调'选对人'和'被拒后不放弃'。"
  - "支持网络的建立需要策略性规划，不是'告诉越多人越好'——错误的披露可能导致信息泄露给施暴者、引发'善意但有害'的介入、或造成二次伤害。"
```

**limits:**
```
limits:
  - "不能保证每一个亲友都会给出有效支持——人际支持存在不确定性"
  - "亲友的非专业介入（如主动去找施暴者理论）可能激化冲突，需要明确告知对方不要自行行动"
  - "施暴者通常在外人面前表现良好，部分亲友可能不相信受害者的描述"
  - "本胶囊仅覆盖'向身边的人开口和建立支持网络'这一步——更系统的安全规划、法律保护、经济准备需转其他胶囊"
  - "不能代替用户做决定——向谁说、说多少、何时说，最终由用户自己判断"
```

**content:**
```
content:
content:

  - theme: "打破'不够严重'的心理门槛"
    points:

         哪怕只是打一个电话、了解自己有什么选项——

      - "这些机构有法定职责介入家暴——

    points:
      - "找妇联或社区求助不会自动启动刑事程序。

      - "你可以在联系时明确说'我现在不希望有人联系他/她，我只是想了解信息'——
         大多数机构会尊重这个要求。"

      - "你本人无法求助时（被控制、被威吓），
         近亲属、公安机关、妇联、居委会、村委会、救助管理机构

  - theme: "了解机构的边界"

      - "妇联和社区是可信赖的服务机构，

         不能替代公安、法院等公权力机构的干预。

      - "不是每个部门都清楚自己的职责，

      - "如果第一个渠道态度敷衍或只会劝和，

      - "面对不理解或不耐烦的工作者，
         可以'温柔而坚定'地沟通——
         态度是温柔的，但求助的决心是坚定的。
         如果遇到推诿，不要放弃，更不要怀疑自己。""

    且可在后续保护令程序中作为协助执行单位"
  - "向法律援助中心求助：经济困难的家暴受害者可申请免费法律援助，
    由援助律师代理保护令申请、离婚诉讼等法律程序"
  - "向救助管理机构/庇护站求助：各地民政部门设有家暴受害者临时庇护场所，
    提供短期食宿、基本生活保障和转介服务；
    入住通常免费，地址对外保密"
  - "向医疗机构求助：就医时主动告知医护人员受伤原因，
    要求病历中注明'家庭暴力致伤'；
    医疗机构发现家暴应按规定向公安机关报告"
  - "多机构联动的意义：单一机构的保护可能存在盲区，
    同时让公安、妇联、社区知晓情况，可以形成多方监督网络，
    提高施暴者的违法成本"


  # ═══ 三、《彩虹暴力终结所——性与性别少数群体反家暴手册》
  #      — 各机构法定职责与求助路径（p51）═══

  - "公安机关的法定职责：接到家暴报警后必须及时出警、制止暴力、
    调查取证、协助受害人就医；对加害人给予批评教育或出具告诫书；
    情节严重的依法给予治安管理处罚"
  - "人民法院的法定职责：受理保护令申请后72小时内（紧急情况24小时内）
    作出裁定；裁定后送达双方及公安、社区；执行保护令并对违反者予以惩处"
  - "民政部门的法定职责：设立临时庇护场所提供庇护服务；
    对无民事行为能力或限制民事行为能力的受害者
    （如未成年人、精神障碍者）提供必要的安置和救助"
  - "妇女联合会的法定职责：接受家暴投诉和反映，提供维权咨询和服务；
    可代为申请保护令；可向公安、法院等相关部门发出建议、督促落实"
  - "居（村）民委员会的法定职责：协助调解家庭纠纷；
    发现家暴或接到求助后应予以劝阻和调解；
    协助执行法院签发的人身安全保护令"
  - "性少数群体向上述机构求助时可能面临额外顾虑
    （如担心被歧视、关系不被认可等），
    可优先联系对性少数友好的公益法律组织获取支持和陪同，
    由其协助与正式机构对接"

**limits:**
```
limits:
  - "居委会/妇联的调解不具有法律强制力，施暴者可能不配合"
  - "庇护所资源各地差异较大，部分地区暂无独立庇护场所或床位有限"
  - "不同地区、不同工作人员的业务水平和态度存在差异——用户可能遭遇敷衍或不当劝和"
  - "妇联和社区不能替代公权力机构的干预——他们提供的是协调、服务性工作"
  - "社区/机构的介入是'第一道支持'，复杂法律问题需转专业律师，心理创伤需转专业咨询"
  - "不能保证机构介入一定能阻止暴力再次发生——机构支持是安全网的一部分，不是万能解决方案"
  - "不能代替用户做决定——是否求助、求助到什么程度、是否接受调解，最终由用户自己判断"
  - "该司法解释仍未将经济控制和性暴力明确列入家庭暴力范围，但以'等'字赋予法官自由裁量权"
```
