N5d：是否离婚

2. 触发条件
triggers:
  - "我要不要离婚" / "我想离但又害怕" / "我不确定该不该走"
  - "离婚是不是太极端了" / "为了孩子我是不是应该忍"
  - "离了我怎么活" / "我什么都没有，离了怎么办" / "我想离但我没有自己的收入"
  - "他说他会改的" / "他有时候对我很好"
  - "离婚之后孩子怎么办" / "离婚之后他还会通过孩子来找我吗"
  - "我怕离了之后他报复我" / "他威胁我说离婚就杀了我" / "离了婚是不是就彻底安全了"
  - "我不确定我受的算不算够严重" / "离婚是不是代表我失败了"
  - "我想过离婚但不知道从哪开始" / "我还爱他，但我也很害怕"

  以下场景符合其一即可调用：
  - 用户在考虑离婚，但因恐惧、内疚、经济依赖、施暴者威胁或信息不足而犹豫。
  - 用户对离婚有误解（如认为必须对方同意、必须有证据、家暴不够严重不能离）。
  - 用户对离婚有不切实际的期待（如离婚后问题自动消失）或恐惧（如必然一无所有）。
  - 用户在"走"和"留"间摇摆，需要梳理真实需求和现实条件，而非要人替其做决定。

3. 适用规则
do_not_use_when:
  - 用户正遭遇暴力或处于紧急人身危险中（转NC紧急求助）
  - 用户已明确决定离婚，仅需了解具体法律流程（转N5e）
  - 用户有自伤风险或表达活不下去（转危机干预）
  - 用户仅需情绪疏导而非决策支持（转情绪支持）
  - 仅为一般婚姻冲突，不涉及暴力/控制/虐待（超出服务范围）

3.5 AI 立场与角色定位
ai_stance:
  core_principle: "AI 不替用户做决定，尊重其在充分信息下的自主选择。"
  prohibited:
    - 不推动离婚：决定权在用户。
    - 不做道德评判：绝不暗示"不离婚就是软弱/不够清醒"。
    - 不流露失望：对用户选择暂时留下保持尊重。
    - 不美化离婚：不将离婚简单等同于"解脱"或"问题自动消失"。
  core_task: "帮用户厘清真实需求与现实条件，拆解决策维度。"
  stance_on_staying: "留下不等于接受受害，可同步建立安全网和积累资源。"
  stance_on_leaving: "离开不等于问题自动解决，需建立准确预期与安全规划。"

4. 认知引导（recognize）
recognize.purpose:
"帮助用户理解：决定权在自己，没有标准答案。协助看清被情绪和误解遮蔽的现实，打破认知障碍，建立对离婚的合理预期。"

content:
  - theme: "你不需要现在就做决定"
    points:
      - "决定可以等，准备不需要等。了解选项和评估处境本身就是有意义的。"
      - "犹豫不代表软弱，离开涉及风险，反复摇摆是合理的自我保护。"

  - theme: "厘清真实需求：你想通过离婚解决什么问题"
    points:
      - "将笼统的'想离婚'拆解为具体诉求（如停止暴力、保护孩子、经济自主）。"
      - "逐一评估离婚能否实现目标：有些能解决（结束婚姻），有些需配套措施（如防探视骚扰需保护令），有些需提前规划（如共同债务、经济独立）。"

  - theme: "打破让你'不敢离'的认知障碍"
    points:
      - "'他有时很好/说会改'：间歇性示好是暴力循环的特征，缺乏长期专业矫治的承诺不可信。"
      - "'为了孩子忍'：目睹家暴对孩子伤害极大，无暴力的单亲环境优于暴力的完整家庭。"
      - "'经济活不下去'：法律保障财产分割、抚养费和赔偿，经济准备可悄悄开始。"
      - "'他威胁我'：威胁是控制手段，说明更需离开，但需先做安全规划。"
      - "'不够严重/代表失败'：持续的恐惧和痛苦就足够构成理由；离开伤害是勇敢而非失败。"

  - theme: "打破让你觉得'离了就好了'的不切实际期待"
    points:
      - "'离婚就解脱了'：离婚是婚姻终点，但不一定是暴力终点。需警惕对方借探视孩子继续骚扰控制。"
      - "'离了就彻底安全了'：分居/离婚期往往是暴力升级高危期，安全规划重于仓促行动。"
      - "对经济完全依赖者：盲目离婚可能带来更深困境。先做经济与支持网络准备是策略，不是怯懦。"

  - theme: "离开与留下的现实考量"
    points:
      - "离开是过程而非瞬间：可在对方不知情下完成信息收集与资源积累。"
      - "留下也可以清醒：留下不等于接受暴力，可同时持续建立安全网和保存证据。"

5. 行动指引（act）
act.purpose:
"帮用户建立决策框架：评估处境、厘清需求、盘点资源、了解选项，为自主选择做好准备。"

steps:
  - step: "评估当前处境"
    detail: "诚实自问：暴力是否升级？是否持续恐惧？他有无致命威胁或全面控制？孩子是否受影响？"
  - step: "厘清真实需求"
    detail: "列出想通过离婚解决的具体问题，评估离婚是否真能解决，或需要哪些配套措施（如保护令、抚养安排、收入规划）。"
  - step: "多维度评估准备程度"
    detail: "盘点四大维度：①人际支持网络；②独立养育能力；③经济独立程度；④离婚后暴力再发风险。条件不足可提前规划补强。"
  - step: "了解法律选项"
    detail: "中国诉讼离婚无需对方同意，家暴是法定离婚事由。你有权争取财产、赔偿、抚养权及保护令（转N5e）。"
  - step: "评估离开风险"
    detail: "预判对方知情后的反应及暴力升级可能，确保有安全去处和防范措施。"
  - step: "悄悄开始准备（无论走留）"
    detail: "备份重要证件、保存家暴证据（转N4）、摸清家庭财产负债、建立独立收入来源、寻找信任支持（转N3c-1）。"

common_questions:
  - question: "离婚需要对方同意吗？"
    answer: "协议离婚需要。若对方不同意，可直接向法院起诉离婚。家暴是法定事由，查实后法院应判离。"
  - question: "第一次起诉法院会判离吗？"
    answer: "若无充分证据，第一次可能不判离；但有家暴证据应判离。即使未判，6个月后二次起诉通常会判离。"
  - question: "有30天离婚冷静期吗？"
    answer: "冷静期仅限协议离婚。家暴案件建议直接诉讼离婚，不受冷静期限制。"
  - question: "他说离婚就不让我见孩子/要自杀"
    answer: "抚养权按'最有利未成年人'判定，家暴是不利因素；探视权受法律保护。自杀威胁是情感操控，不应成为你的枷锁。"
  - question: "我没收入，也没家暴证据，能离吗？"
    answer: "无证据也可起诉。无收入者有权分割共同财产并申请赔偿/补偿。现在开始收集证据和做经济准备完全来得及。"
  - question: "离婚后他借探视孩子骚扰我怎么办？"
    answer: "这是真实风险。可向法院申请中止探视、要求第三方监督探视，或申请保护令禁止骚扰。"
  - question: "我经济不独立又没朋友，离了会不会更惨？"
    answer: "无准备的离婚确实可能带来新困境。这说明你'需要先做准备'而非'不能离'。可先了解法定权利、寻找收入机会、建立支持网络，条件成熟后再行动。"

safety_note:
  - "绝不在无安全计划或激烈冲突时当场提离婚，极易引发暴力升级。"
  - "若对方已有致命威胁，提离婚前应先申请人身安全保护令。"
  - "所有查询（含搜索记录）和准备工作必须在对方不知情下进行。"

6. 依据与限制（ground）
ground.purpose:
"提供法律依据，说明实务困难。"

legal_basis:
  - "《民法典》：第1079条（家暴为法定离婚事由）、第1077条（冷静期仅限协议）、第1091条（家暴损害赔偿）、第1087/1084条（财产与抚养权原则）。"
  - "《民法典婚姻家庭编解释(一)》第63条：首次未判离分居一年后再次起诉应判离。"
  - "《反家庭暴力法》：保护令制度。"

practice_basis:
  - "受害者平均尝试7次才能最终离开，反复是正常过程。"
  - "分居/离婚阶段是暴力致死高危期，'安全离开'重于'立刻离开'。"
  - "高达40%-70%家暴案在离婚后暴力未完全停止，需配套保护措施。"
  - "对基层妇女，有准备的离开其后续生活质量远优于仓促离开。"
  - "施暴者无专业矫治的改变承诺极难兑现；目睹家暴对儿童伤害极大。"

limits:
  - "本胶囊不替用户做决定，不推动离婚，不对经济依赖者简单劝离。"
  - "离婚不是万能方案，不能自动消除后续骚扰风险。"
  - "仅覆盖'决策阶段'，具体诉讼流程转N5e，心理创伤需专业心理咨询。"
