# K1: 家暴常见误区

capsule_name: "家暴常见误区"
status: "draft"
triggers:
  - "他说打我是因为我惹他生气了"
  - "没动手就不算家暴吧"
  - "冷暴力也算家暴吗"
  - "他是为了我好才限制我社交"

use_when:
  - "用户对什么是家暴存在认知偏差"
  - "用户试图合理化对方的控制行为"
  - "用户未意识到自身处于受控状态"

do_not_use_when:
  - "用户正处于遭受身体暴力导致的急性创伤中"
  - "用户表达了自残或自杀意念"
  - "用户处于无法与外界沟通的完全受控状态"

recognize.purpose: "纠正受害者对暴力的刻板印象，确认任何形式的非法侵害都不应被合理化。"
content:
  - "暴力不仅包含身体伤害，还包括精神、经济及性暴力"
  - "对方的施暴理由永远不能成为其推卸责任的借口"
  - "任何形式的孤立、监控与羞辱都是对人格的侵害"
  - "受害者没有义务承担施暴者的情绪后果"

act.purpose: "引导用户通过重新界定认知，提升自我保护意识并寻求专业援助。"
steps:
  - "识别并记录日常中发生的控制行为（如检查手机、贬低价值、控制开支）"
  - "客观评价对方的借口，明确这些行为已触犯法律或道德底线"
  - "联系身边的支持者或专业组织，验证自身所处环境的安全性"
  - "在保证隐私的前提下，开始收集并记录每一次暴力发生的证据"

scripts:
  - "我以前以为只有身体受伤才算家暴，现在我意识到这种持续的贬低和控制也是暴力的一部分。"
  - "我不想再因为害怕和妥协而继续维持这段关系，我想了解如何保护自己。"

safety_note:
  - "如果对方正在监控你的通讯工具，请务必使用后删除记录或寻找安全设备联系。"
  - "在进行心理评估和识别时，请确保你处于对方无法监视的安全空间。"

ground.purpose: "基于法律实务与反家暴原则，厘清暴力定义，管理用户维权预期。"
legal_basis:
  - "《中华人民共和国反家庭暴力法》第二条（关于家庭暴力的定义）"
  - "《中华人民共和国民法典》第一千零四十二条（禁止家庭暴力）"
practice_basis:
  - "源众性别暴力介入中心《家庭暴力受害者自我保护手册》（2025版）"
limits:
  - "仅提供认知引导与风险防范建议，不能代替法律诉讼或心理治疗"
  - "不对用户的具体离婚结果或财产分割结果提供保证"
