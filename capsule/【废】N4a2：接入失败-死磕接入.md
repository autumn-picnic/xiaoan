# 1. 基础信息
capsule_name: "被拒后仍需基层接入的死磕策略"
status: "draft" 

# 2. 触发条件
triggers:
  - "虽然居委会之前推诿，但我现在需要他们陪我去物业调监控取证"
  - "邻居听到了打骂声但怕事不肯作证，必须让居委会出面走访"
  - "我现在没钱住酒店也无处可去，必须让基层帮我联系庇护所"
  - "我需要对施暴者形成日常威慑，想让网格员天天上门走访敲打他"

# 3. applicability 适用规则
use_when:
  - "用户虽然此前遭遇基层推诿，但目前面临特定的‘非官方不可’的实操困境（如调监控、申请官方庇护、日常属地监控）"
  - "用户暂无迫在眉睫的切肤之痛（若正在挨打优先引导拨打110）"
  - "用户有心理建设，愿意配合话术对基层进行‘行政与考核施压’"

do_not_use_when:
  - "面临紧急人身危险（直接右转走110流程）"
  - "施暴者与基层工作人员存在极深的私人关系，强行接入极易导致信息泄露和报复（此时应放弃基层，走公安/法院人身安全保护令赛道）"

# 3. recognize 认知引导
recognize.purpose: "纠正‘基层没用就彻底放弃’的误区，帮助用户认识到基层组织是搬不走的‘地头蛇’，在调取证据、属地威慑和民政救助上具有不可替代的行政功能，必须反向利用其‘怕出事’的心理为我所用。"

content:
  - "居委会和街道办最怕的不是法条，而是‘辖区内出恶性大事’和‘绩效考核被一票否决’。"
  - "不要和办事员纠缠‘家务事’的定性，直接谈‘属地安全责任’和‘重大恶性案件倒查’。"
  - "他们不需要帮你打官司，但他们必须帮你开门（调监控）、引路（进庇护所）、盯人（网格员上门）。"

# 4. act 行动指引
act.purpose: "指导用户精准切入基层的核心痛点，通过‘战略回马枪’逼迫基层履行特定场景下的协助与救助职责。"

steps:
  - "**策略一：证据保全死磕（调监控/做笔录）**"
    "以‘监控覆盖通常只有7天，因基层推诿导致证据灭失需承担法律责任’为由，要求居委会或街道司法所工作人员陪同前往物业调取监控，或由居委会出面做邻居的《走访谈话笔录》。"
  - "**策略二：日常施压死磕（网格员频繁上门）**"
    "避开普通办事员，带上亲友直接找【社区党支部书记】或【街道综治办】。明确表示不求调解，只求‘网格员密集上门走访警告’，将行政视线死死盯在施暴者身上。"
  - "**策略三：资源断绝死磕（强制转介庇护所）**"
    "明确告知拒绝调解，由于无家可归，要求其履行民政救助职责。若对方推托没床位，需要留下求助痕迹（录音、微信截图、接访登记表），以此为据直接去区民政局督办。"

scripts:
  - "调监控/走访话术：“根据《反家暴法》，居委会有协助取证的义务。监控7天就会自动覆盖。如果因为社区不配合导致关键证据灭失、我在法庭上败诉，倒查责任时，我今天的到访记录和录音就是社区推诿塞责的铁证。请立刻派网格员陪我去物业。”"
  - "要求上门走访话术（对书记/综治办）：“书记，我现在不要求你们调解离婚。我只要求网格员明天去我家走访，当面警告他社区已经盯上他了。如果你们连上门劝阻都不去，他在辖区里闹出刑事重案，大家今年的平安建设考评和绩效就彻底泡汤了。您看怎么安排？”"
  - "要庇护所转介单话术：“我现在人身安全正面临极大危险。请社区立刻帮我对接区救助站或反家暴庇护所。如果本区没床位，请协调跨区安置。如果你们拒绝联系，请现在给我开一份《无法提供紧急庇护的说明》，我拿着这个直接去区民政局。”"

safety_note:
  - "借力打力：去找基层死磕上述诉求时，尽量不要单枪匹马，带上靠谱的亲友、律师或志愿者陪同，人多势众往往能极大加速基层的办事效率。"
  - "绝不挑衅：在外施压要硬，但回到家面对施暴者要保持低调，绝不要拿‘社区已经盯上你了’去挑衅对方，防止其情绪失控现场施暴。"

# 5. ground 依据与限制
legal_basis:
  - "《中华人民共和国反家庭暴力法》第十三条：有关单位接到投诉、反映或者求助后，应当给予帮助、处理。"
  - "《中华人民共和国反家庭暴力法》第四条：县级以上人民政府负责妇女儿童工作发展的机构，负责组织、协调、指导、督促有关部门做好反家庭暴力工作。"

limits:
  - "基层协调庇护所或调取监控可能存在跨部门沟通的行政时滞，若遭遇极端紧急危险，请立刻放弃本方案，直接报警或物理撤离。"
