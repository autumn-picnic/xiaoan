# applicability

## use_when

- 用户已决定离婚，需了解具体法律流程和操作步骤
- 用户需在家暴背景下争取有利的抚养权、财产分割、损害赔偿
- 用户在诉讼中遇到困难（对方不同意、不判离、转移财产、诉讼期间受骚扰）
- 用户经济困难需要法律援助

## do_not_use_when

- 正在遭遇暴力/紧急人身危险 
- 仍在犹豫是否离婚 
- 自伤风险或表达活不下去 
- 不涉及家暴的一般离婚咨询

# recognize

- 家暴是法定离婚事由 + 损害赔偿依据：《民法典》第1079条明确家暴为感情破裂的法定证据，第1091条赋予受害方离婚损害赔偿请求权（可随诉讼一并提出，也可离婚后一年内单独起诉）
- 认定存在家暴应尽快判决离婚：审理指南第17条规定，认定存在家暴且一方坚决要求离婚的，法院应尽快调解或判决离婚，避免久拖不决导致更严重伤害
- 财产和抚养权对受害方有利：财产分割照顾无过错方（第1087条），施暴行为不利于获得抚养权（"最有利于未成年子女"原则），财产转移多发生在诉讼前，应尽早申请财产保全
- 不需要对方同意但第一次不一定判离：对方拒绝签字、拒绝出庭，法院可缺席判决；即使认定家暴仍有约24%首次未判离；6个月后可再次起诉，不必气馁
- 判不离后再施暴可构成再诉"新情况、新理由"：法释〔2022〕17号第十一条规定，判决不准离婚或调解和好后，被申请人违反人身安全保护令再次施暴的，可据此重新起诉，无需等待6个月
- 证据是最关键的环节：实证数据显示仅29.8%的原告提交了证据，超半数为孤证；认定家暴离婚支持率75.76%，未认定仅10%；现在开始保存证据还来得及
- 诉讼期间是暴力升级高危期（"分手暴力"）：离婚/分手请求常刺激加害人的控制欲，应同步申请人身安全保护令，不要在无安全保障时告知施暴者离婚计划
- 家暴背景下协议离婚风险高：施暴者控制力可能延伸到谈判桌上，诉讼离婚有法官把关更常见也更安全
- 离婚后骚扰仍可申请保护令或报警处理，保护措施不因进入离婚程序或离婚完成而放松

# act

## 诉讼离婚操作流程

1.前期准备

- 确认管辖法院（通常为被告住所地基层法院）
- 收集固定证据（转证据收集与固定模块）
- 梳理财产状况（尽早掌握对方财产线索，防止转移）
- 明确抚养权诉求
- 咨询律师（经济困难可申请法律援助，拨打12348，家暴受害者多地免审查经济条件）
- 考虑同步申请人身安全保护令

2.立案

- 准备材料：起诉状、身份证、结婚证、证据材料
- 向法院提交（可现场或网上立案）
- 诉讼费50–300元，经济困难可申请缓交/减免
- 如担心转移财产，立案时同步申请财产保全

3.等待开庭

- 法院送达传票给被告
- 继续补充证据
- 与律师沟通庭审策略

4.开庭审理

- 法庭调查 → 举证质证 → 法庭辩论 → 最后陈述
- 法院通常先调解：条款公平可接受，不公平有权拒绝

5.判决

- 判决离婚 → 执行
- 判决不离婚 → 6个月后可再次起诉（若对方违反保护令再施暴，可认定为"新情况、新理由"立即再诉）
- 不服判决可在15日内上诉

6.离婚损害赔偿

- 提出时机：建议随诉讼一并提出
- 举证要求：证明对方存在家暴行为、家暴导致离婚、你遭受的损害
- 赔偿金额：法律未规定固定金额，实务中精神损害赔偿数千至数万元不等

7.抚养权争取

- 核心原则："最有利于未成年子女"
- 2周岁以下：原则母亲抚养
- 2–8周岁：综合考量
- 8周岁以上：尊重孩子意愿
- 家暴因素：施暴行为不利于获得抚养权
- 探视权风险：可请求"监督探视"（第三方在场或指定场所）降低骚扰风险
