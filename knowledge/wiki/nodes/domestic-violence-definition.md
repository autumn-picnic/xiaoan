# 家庭暴力定义

## 原子结论

家庭暴力不是“家务事”。法律定义包括家庭成员之间通过殴打、捆绑、残害、限制人身自由，以及经常性谩骂、恐吓等方式实施的身体、精神等侵害行为。

家庭成员以外共同生活的人之间实施的暴力行为，参照反家庭暴力法规定执行。

## 适用边界

- 法条明确列举的是典型方式，不应把用户处境机械限制在列举词汇里。
- 小安不能替代法院或执法机关作出最终法律认定。
- 未成年人、老年人、残疾人、孕期和哺乳期妇女、重病患者遭受家庭暴力时，应当给予特殊保护。

## 可用于回答

- “这算家暴吗？”
- “他只是骂我/恐吓我，也算吗？”
- “我们不是结婚关系，只是一起生活，能不能适用反家暴法？”

## 最高法典型案例补充认定（2023-2026）

下表汇总最高法典型案例对《反家庭暴力法》意义上的暴力形态所作的认定。刑事案件中的罪名、犯罪主体和因果关系不在本定义节点内展开，避免跨法域混同。

| 类型        | 认定要点                                      | 来源              |
| --------- | ----------------------------------------- | --------------- |
| 语言暴力      | 经常性谩骂、侮辱、诋毁，造成精神伤害；可支持离婚损害赔偿              | 2026年3月案例1         |
| 限制正常社会交往  | 通过殴打/辱骂方式禁止配偶与他人接触，侵害人身自由权、名誉权            | 2026年3月案例2         |
| 经济控制＋阻止就医 | 掌控家庭收入一方拒绝支付医疗费、强制出院，形成持续精神压制；与殴打构成"叠加家暴" | 2026年3月案例3         |
| 自伤、自残威胁   | 以持刀自残等方式制造恐惧、迫使配偶屈服并维持控制，可构成精神暴力           | 2025年11月案例5         |
| 暴力管教子女    | 反复殴打、责骂并造成身体伤害、恐惧焦虑和重度抑郁等严重身心损害，已超出正常教育限度，不得以“管教”为由免责 | 2025年11月案例6         |

注意：小安可援引上述认定解答"这算家暴吗"类问题，但仍不能替代法院最终认定。
