# 受暴者反杀施暴人的刑事处理

## 原子结论

长期遭受严重家庭暴力的受害者，在激愤、恐惧状态下为防止再次受暴或者摆脱家暴而杀害施暴人的，人民法院可结合案件起因、作案动机、被害人过错、犯罪手段等因素，认定为故意杀人罪“**情节较轻**”（刑法第232条），并结合自首、认罪认罚、被害人亲属谅解等情节从宽处罚。最高人民法院发布的两个典型案例形成了一致裁判示范。

> 注意：这不是鼓励"以暴制暴"。受暴者应优先通过法律途径（报警、申请保护令、起诉离婚等）维权。本节点仅描述已发生极端后果时的司法处理原则。

## 认定"情节较轻"的典型情形

| 要素 | 说明 |
| --- | --- |
| 前提 | 长期、严重的家庭暴力史（非偶发矛盾） |
| 主观状态 | 为防止再次遭受家暴或者摆脱家暴；是否存在激愤、恐惧及报复心理须结合个案判断 |
| 被害人过错 | 施暴方对案件起因具有明显单方过错 |
| 从宽情节 | 自首、认罪认罚、被害人亲属谅解 |

## 适用边界

- "情节较轻"是量刑情节，不是无罪。受暴者仍将被定罪，只是刑期较轻。
- 小安不向用户建议"以暴制暴"路径；遭受家暴应优先援引 `personal-safety-protection-order`、`divorce-and-dv` 等合法救济途径。
- 本节点来源仅为法院典型案例（court_cases），不是法律条文；具体案件需结合律师意见。
