# 地方性法规细化对比（18省/市/区）

> 本页 ingest 全部省级反家暴条例/实施办法，按“相对国家《反家庭暴力法》的细化点”做横向对比。它们与国家法是 `localizes` 关系：在不抵触上位法的前提下细化定义、处置、告诫书、保护令协助执行等。
>
> 广东省已单独建节点 [[guangdong-implementation]] 作为 worked example；本页覆盖其余 17 省并把广东并入对比。
>
> 边界：本页是地方差异的对照索引，**仅适用各自行政区域**；个案以当地现行条例原文为准。`needs-review`：矩阵基于对各省条文关键词与定义条款的抽取，具体条号与表述需法律专业人士逐省复核。节点引用用行内代码，避免本页成为 graph hub。

## 一、共性细化（多数省份共有）

| 细化点 | 相对国家法的变化 | 代表省份 |
| --- | --- | --- |
| 网络手段 | 明确通过网络等手段实施恐吓/威胁/精神侵害属于家暴 | 全部 18 省 |
| 经济侵害/控制 | 把“非正常经济控制、剥夺财物”列为独立家暴类型 | 安徽、山东、山西、广东、河南、甘肃、辽宁、重庆、陕西、黑龙江 |
| 冻饿 | 定义明确列举“冻饿”等身体侵害 | 广东、甘肃、贵州、辽宁、重庆、青海、黑龙江 |
| 目睹家暴未成年人 | 明确“目睹家庭暴力的未成年人也是受害人”，给予保护/心理辅导 | 云南、安徽、广东、新疆、河北、海南、湖北、甘肃、辽宁、重庆、陕西、青海、黑龙江 |
| 跟踪/骚扰/散布隐私 | 精神侵害类型细化 | 多数省份 |

## 二、处置与救济细化

| 细化点 | 相对国家法的变化 | 代表省份 |
| --- | --- | --- |
| 110接处警/出警时限 | 家暴警情纳入110，明确出警与处置步骤 | 云南、安徽、山东、广东、河南、海南、湖北、新疆、重庆等 |
| 告诫书强制情形 | 规定“应当出具告诫书”的情形（特殊群体、再犯、受害人要求等） | 广东、山东、重庆、辽宁、黑龙江等 |
| 远离令/迁出令 | 保护令协助执行细化，部分增加“责令远离场所” | 云南、山东、广东、新疆、河北、海南、湖北、辽宁、重庆、青海、黑龙江 |
| 代为申请扩展 | 扩展可代为申请保护令的主体/情形 | 云南、山东、广东、新疆、河北、河南、海南、湖北、辽宁、重庆、陕西、贵州 |
| 强制报告制度 | 明确“家庭暴力强制报告制度”并禁止打击报复报告人 | 山西、湖南、重庆、陕西（重庆并明确加害人不得打击报复） |
| 临时庇护 | 庇护场所设立、入住凭证（部分凭告诫书）等细化 | 多数省份 |
| 反家暴热线 / 信息共享 | 设立热线、建立部门信息共享/统计机制 | 云南、安徽、山西、河南、重庆、贵州、青海、甘肃、海南、湖南、黑龙江等 |

## 三、标志性条款（已核对原文）

- **重庆**（第2条）：列“非正常经济控制、剥夺财物等经济侵害行为”，网络手段实施第三/五/六项侵害属家暴；“目睹家庭暴力的未成年人”纳入心理辅导对象；明确强制报告且加害人不得打击报复。
- **甘肃**（第3条）：定义按“身体暴力、精神暴力、性暴力、经济控制”分类；明确“目睹家庭暴力的未成年人也是家庭暴力受害人”。
- **黑龙江**（第3条）：定义分类含经济控制；把“迫使目睹暴力”列为精神侵害。
- **山西**（第13条）：明确“建立家庭暴力强制报告制度”。

## 四、机制关系

本页整体对国家法节点构成 `localizes` 关系（与 [[guangdong-implementation]] 同类）：

- `local-regulations-comparison` --localizes--> `domestic-violence-definition`（经济/网络/冻饿/目睹等定义扩展）
- `local-regulations-comparison` --localizes--> `public-security-response-duty`（110接处警、告诫书强制情形、强制报告）
- `local-regulations-comparison` --localizes--> `personal-safety-protection-order`（远离/迁出、代为申请、协助执行）
