# 最高人民法院反家暴典型案例汇编（2023-2026）

共33个典型案例，跨越2023年6月、2023年11月、2024年11月、2025年11月、2026年3月五批发布。按法律主题分组，每案注明批次/编号与核心判旨，以便查找与援引。

---

## 一、家庭暴力形式扩展认定

参见 [[domestic-violence-definition]]。

| 案例               | 核心判旨                                  |
| ---------------- | ------------------------------------- |
| 2026案例一（赵某诉张某）   | 经常性谩骂、侮辱人格 = 精神家暴，可据此支持离婚损害赔偿         |
| 2026案例二（王某保护令）   | 长期以殴打/辱骂方式限制配偶正常社会交往 = 家暴（侵害人身自由、名誉权） |
| 2026案例三（陈某保护令）   | 掌控家庭收入一方拒绝就医 + 殴打 = 叠加家暴；经济控制属家暴本质特征  |
| 2025案例6（李某抚养）    | 直接抚养人的暴力管教 = 家庭暴力，不得以"管教"为由免责         |
| 2025案例5（鲁某保护令）  | 以自伤、自残相威胁并制造恐惧以控制配偶，可构成精神暴力          |
| 2024案例四（刘某坤虐待重婚） | 非婚同居关系中，具有共同生活基础事实的双方属"家庭成员"，虐待成立     |
| 2023Jun案例3（故意伤害） | 管教子女非家暴理由，对子女实施家暴当场致死应认定故意伤害罪         |

## 二、人身安全保护令——申请、执行与违反后果

参见 [[personal-safety-protection-order]]、[[liability-ladder]]。

| 案例 | 核心判旨 |
| --- | --- |
| 2026案例四（张某违反保护令） | 违反保护令多项禁令（再次施暴+骚扰），情节较重未达刑事 → 拘留15日 |
| 2025案例5（鲁某保护令） | 一站式联动闭环干预机制：法院+派出所+居委会+妇联多部门协同执行 |
| 2024案例五（王某辉拒不执行） | 违反保护令情节严重 → 构成**拒不执行裁定罪**，判处有期徒刑8个月（刑事责任） |
| 2024案例三（梁某伟） | 受害者报警后"法院+妇联"联动，签发保护令并判离婚，联系心理咨询 |
| 2023Nov案例一（蔡某某） | 暴力抢夺子女+当面殴打母亲 → 子女可申请保护令（间接受害人） |
| 2023Nov案例二（唐某某） | 幼儿园强制报告 → 妇联代为申请 → 法院签发保护令；责令施暴人接受心理辅导 |
| 2023Jun案例5（回访督促） | 保护令签发后法院应回访、督促被申请人履行 |
| 2023Jun案例6（在线审理） | 全流程在线审理人身安全保护令 |
| 2023Jun案例7（同居结束后） | 同居关系结束后，受暴方仍可向法院申请人身安全保护令 |

## 三、施暴方刑事责任（重罪）

参见 [[liability-ladder]]。

| 案例 | 核心判旨 |
| --- | --- |
| 2025案例1（牟某虐待） | 精神虐待致被害人自杀死亡 = 虐待罪，情节恶劣（同居男友属"家庭成员"） |
| 2025案例3（许某某） | 施暴人为维持控制持刀捅刺受暴人胸部，构成故意杀人罪（未遂）；延迟控告不降低受暴人陈述可信度 |
| 2024案例一（谢某宇杀人） | 因不满配偶起诉离婚预谋杀人 → 故意杀人罪，核准死刑 |
| 2024案例四（刘某坤虐待重婚） | 虐待+重婚数罪并罚；哺乳期妇女和未成年人受特别保护 |
| 2023Jun案例1（陈某某杀人） | 酗酒自陷行为不减轻刑责；证人证言可构成认定家暴的主要证据，不得认定初犯 |
| 2023Jun案例3（管教致死） | 对子女家暴当场致死 = 故意伤害罪；管教不是免责理由 |

## 四、受暴者反抗杀施暴人的刑事处理

参见 [[dv-survivor-homicide]]。

| 案例 | 核心判旨 |
| --- | --- |
| 2024案例二（赵某梅） | 长期严重家暴 → 被迫杀施暴人，认定故意杀人"**情节较轻**"，结合自首认罪认罚从宽处罚 |
| 2023Jun案例2（姚某某） | 受暴妇女不堪忍受杀施暴人，认定"情节较轻"；系首例适用两高两部《反家暴意见》第20条的案件 |

## 五、未成年子女保护

参见 [[child-witness-victim]]、[[mandatory-reporting]]。

| 案例 | 核心判旨 |
| --- | --- |
| 2025案例4（任某强奸猥亵） | 未成年被害人陈述具有"非亲历不可知细节"时，排除诱证/诬告可能并结合其他证据相互印证后，一般应采信 |
| 2025案例8（纪某抚养） | 施暴方一般不宜直接抚养未成年子女 |
| 2023Nov案例一（蔡某某抢夺） | 暴力抢夺、藏匿未成年子女 + 目睹家暴 → 子女为家暴受害人 |
| 2023Nov案例二（唐某某） | 幼儿园强制报告义务；全社会合力救护被家暴未成年人 |
| 2023Nov案例三（刘某某） | 离婚纠纷中施暴方不宜直接抚养未成年子女 |
| 2023Nov案例四（学校强制报告） | **学校**发现未成年人遭受或疑似遭受家暴，应履行强制报告义务 |
| 2023Nov案例五（暂时变更抚养） | 直接抚养人实施家暴，法院**可暂时变更**直接抚养人 |
| 2023Nov案例六（受教育权） | 父母行为侵害未成年子女受教育权，子女可申请人身安全保护令 |
| 2023Jun案例9（抚养权） | 受暴方过错并非家暴理由，施暴方不宜直接抚养 |

## 六、离婚、撫養與財產权益

参见 [[divorce-and-dv]]。

| 案例 | 核心判旨 |
| --- | --- |
| 2025案例7（许某诉郑某） | 受暴全职家庭妇女：同时支持**家务劳动补偿金**（民法典1088条）+ 离婚损害赔偿 |

## 七、证据认定特则

参见 [[protection-order-evidence]]、[[civil-dv-fact-finding]]、[[sexual-violence-in-family]]。不同程序的证明标准不得混同。

| 案例 | 核心判旨 |
| --- | --- |
| 2025案例2（张某强奸） | 家暴情境下，被害人无明显反抗 ≠ 同意；需充分考虑家暴恐惧心理，必要时听取专业人员意见 |
| 2023Jun案例1（陈某某） | 多名证人证言可构成认定家暴主要证据，无需行政/刑事处罚记录 |
| 2023Jun案例8（离婚案证据审查） | 法院可结合陈述、报警、病历、鉴定及对方解释能力综合认定，并加强职权探知；不能据此把保护令标准自动移植到所有离婚案件 |
