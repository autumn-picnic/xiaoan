## 二、立案审查

| 审查项 | 标准 | 依据 |
| --- | --- | --- |
| 管辖 | 申请人 / 被申请人居住地、家庭暴力发生地的基层人民法院（任择其一） | 第25条 |
| 当事人资格 | 当事人本人可申请；无 / 限制民事行为能力人，或受强制、威吓无法申请的，近亲属、公安机关、妇联、居委会、村委会、救助管理机构可**代为申请** | 第23条 |
| 申请方式 | 书面为原则；确有困难可口头申请，由法院记入笔录 | 第24条 |
| 前置条件 | 无须先提起离婚等诉讼 | 保护令实务（needs-review） |
| 处理时限 | 受理后72小时内作出；情况紧急24小时内 | 第28条 |

⚠️ 常见**不当**拒绝受理理由（实务来源，needs-review）：“下午不受理”“须与离婚一并处理”“至少两份报案回执”——均无法律依据。

## 四、被申请人抗辩（被告抗辩）

实务来源显示，以下抗辩**常见但多不应成立**；列出以便预判与反驳：

| 被申请人抗辩 | 是否成立 | 理由 |
| --- | --- | --- |
| 双方不住在一起 | ✗ 不成立 | 保护令针对“面临现实危险”，不以同住为前提 |
| 已达成调解 / 申请人曾不要求处理 | ✗ 不成立 | 调解不剥夺此后的申请权 |
| 已认错 / 出具悔过书、保证书 | ✗ 不成立 | 道歉保证不视为真心悔改；悔过书反而可作家暴**证据** |
| 暴力频率低 / 非近期发生 | ✗ 不成立 | 针对“现实危险”，不以频繁为前提 |
| 存在其他纠纷（房产、家庭琐事） | ✗ 不成立 | 纠纷的存在不能为暴力辩护 |
| 称申请人有过错 / 有疾病 | ✗ 不成立 | **法释〔2022〕17号第八条**：被申请人认可家暴但辩称申请人有过错的，不影响作出保护令 |

✅ 已解决（原 needs-review）：实务文档所引“第八条”经一手来源核对，对应**法释〔2022〕17号第八条**（已入库），并非误引《反家暴法》第八条。此外《审理指南》第八条另有“家暴非受害人过错”的原则性表述，二者相互印证。

## 五、要件事实（请求权基础分析）★核心

用大陆法系“权利发生 / 妨碍 / 消灭要件”三分法拆解：

### 5.1 权利发生要件（第27条）

| # | 要件 | 争点 | 对应要件节点 |
| --- | --- | --- | --- |
| 1 | 有明确的被申请人 | 通常无争议 | — |
| 2 | 有具体的请求 | 通常无争议 | — |
| 3 | **遭受家暴或面临家庭暴力现实危险** | **核心争点** | `protection-order-element-danger` |

- 要件3的“家庭暴力”含义由 `domestic-violence-definition` 界定（第2条）。
- 证明标准：“较大可能性”（needs-review：法释17号）。

### 5.2 证明结构（证据 → 要件）

证据不直接证明“保护令”，而是先证明**要件3**：

```
公安出警记录 / 告诫书 / 伤情鉴定意见 / 医疗记录 / 证人证言 / 妇联求助记录
        │ provides_evidence_for
        ▼
protection-order-evidence  ──proves──▶  protection-order-element-danger（要件3）
                                                  │ is_element_of
                                                  ▼
                                        personal-safety-protection-order
```

证据来源映射：`public-security-response-duty`、`injury-appraisal-procedure`、`police-dv-handling-workflow`、`support-and-legal-aid`。

### 5.3 权利妨碍 / 阻却要件

第四部分各抗辩多不成立 → 通常**不构成有效阻却**。真正可能阻却的是要件本身不满足（如无证据、无现实危险）。

### 5.4 权利消灭要件

- 保护令有效期不超过6个月，期满消灭；可在失效前申请撤销、变更或延长。 | 第30条
- 复议：不服裁定可于生效之日起5日内申请复议一次，复议期间不停止执行。 | 第31条

### 5.5 违反后果（救济的强制力）

- 违反保护令尚不构成犯罪：训诫，1000元以下罚款或15日以下拘留；构成犯罪追究刑事责任。 | 第34条
- 执行：保护令由法院执行，公安机关及居委会、村委会协助执行。 | 第32条
